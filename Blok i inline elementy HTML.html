Blok i inline elementy HTML

Kozen element HTML mae znachennja vidobrazennja za zamovchuvannjam, zalezno vid typu elementa.
Je dva znachennja vidobrazennja: block i inline.

Elementy blochnogo rivnja

Element na rivni bloky zavzdy pochunaetjsja z novogo rjadka, i brauzery avtomatychno dodajutj probil (pole) pered i pislja elementa.
Element na rivni bloku zavzdy zajmaje vsju dostupnu shyrynu (roztjagujetjsja vpravo i vlivo naskiljky ce mozlyvo).
Dva chasto vykorystovuvani blokovi elementy p i div.
Element p vyznachaje abzac v dokumenti HTML.
Element div vyznachaje podil abo rozdil y elementi HTML.

Hello World! (p)

Hello World! (div)

Element div

ELement div chasto vykorystovuetjsja jak kontejner dlja inshyh elementiv HTML.
Element div ne maje obovjazkovyh atrybutiv, ale style,class i id je zagaljnymy.
Pry vykorystanni razom iz CSS div element mozna vykorystovuvaty dlja stylizaciji blokiv vmisty:

London

London is a capital city of England. It is most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitans.

Element span

Element span - ce vbudovanyj conteiner, jakyj vykorystovuetjsja dlja rozmitky chastyny tekstu abo chastyny dokumenta.
Element span ne maje obovjazkovyh atrybutiv, ale style,class i id je zagaljnymy.
Pry vykorystanni razom iz CSS span element mozna vykorystovuvaty dlja stylizaciji chastyn tekstu:

My mather has blue eyes and my father has dark green eyes
