Атрибути форми введення HTML

У цій главі описуються різні form атрибути для input елемента HTML.

Атрибут form

Атрибут input form визначає форму, до якої належить input елемент.
Значення цього атрибута має дорівнювати атрибуту id елемента form, якому він належить.

<form action="/action_page.php" id="form1">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">

<input type="Submit" value="Submit">
</form>

<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname" form="form1">

Атрибут formaction

Атрибут input formaction визначає URL-адресу файлу, який обраблятиме вхідні дані під час надсилання форми.
Примітка. Цей атрибут перевизначає action атрибут form елемента.
Атирбут formaction працює з такими типами вхідних даних: submit i image.

<form action="action_page.php">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">

<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname">

<input type="submit" value="Submit">
<input type="submit" formaction="/action_page2.php" value="Submit as Admin">
</form>

Атрибут formenctype

Атирбут input formenctype визначає, як мають бути закодовані дані форми під час надсилання (лише для форми із method="post").
Примітка. Цей атрибут замінює атрибут enctype <form> елемента.
Атрибут formenctype працює з такими типами взідних даних: submit i image.

<form action="/action_page_binary.asp" method="post">
<label for="fname">First name</label>
<input type="text" id="fname" name=fname">

<input type="submit" value="Submit">
<input type="submit" formenctype="multypart/form-data"> value="Submit as Myltipart/form-data"
</form>

Атрибут fomrmethod

Атрибут input formmethod визначає метод HTTP для надсилання даних форми до URL-адреси дії.
Примітка. Цей атрибут перевизначає атрибут методу form елемента.
Атирбут formmethod працює з такими типами вхідних даних: submit i image.
Дані форми можна надіслати як змінні URL-адреси (method="get") або як пост-транзакцію HTTP(method="post").
Примітки щодо методу "get":

  • Цей метод додає дані форми до URL-адреси в парах ім*я/значення.
  • Цей метод корисний для надсилання форм, коли користувач хочк додати результат до закладок.
  • Існує обмеження на кількість даних, які ви можете розмістити в URL-адресі (залежить від браузера), тому ви не можете бути впевнені, що всі дані форми буде правильно передано.
  • Ніколи не використовуйте метод "get" для передачі конфеденційної інформації (пароль або іншу конфеденційну інформацію буде видно в адресному рядку браузера).

Примітки щодо методу "post":

  • Цей метод надсилає дані форми як пост-транзакцію HTTP.
  • Надсилання форм за допомогою методу "опублікувати" не можна додавати в закладки
  • Метод "post" надійніший і безпечніший, ніж "get", а "post" не має обмежень щодо розміру
<form action="/action_page.php" method="get">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">

<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname">

<input type="submit" value="Submit using GET">
<input type="submit" formmethod="post" value="Submit using POST">
</form>

Атрибут formtarget

Атрибут input formtarget визначає назву або ключове слово, яке вказує, де відобразити відповідь, отриману після надсилання форми.
Примітка. Цей атрибут замінює цільовий атрибут <fomr> елемента.
Атрибут formtarget працює з такими типами даних: submit i image.

<form action="action_page.php">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">

<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname">

<input type="submit" value="submit">
<input type="submit" formtarget="_blank" value="Submit to a new window/tab">
</form>

Атрибут formnovalidate

Атрибут input formnovalidate визначає, що елемент input не повинен перевірятися під час надсилання.
Примітка. Цей атрибут замінює novalidate form елемента.
Атрибут formnovalidate працює з такими типами введення: submit.

<form action="/action_page.php">
<label for="email">Enter your email:</label>
<input type="email" id="email" name="email">

<input type="submit" value="Submit">
<input type="submit" formnovalidate="formnovalidate" value="Submit without validation">
</form>

Атрибут novalidate

Атрибут novalidate є form атрибут.
Якщо присутнє, novalidate вказує, що всі дані форми не повиння перевірятися під час надсилання.

<form action="/action_page.php" novalidate>
<label for="email">Enter your email:</label>
<input type="email" id="email" name="email">

<input type="submit" value="Submit"> </form>
