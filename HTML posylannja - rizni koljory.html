HTML posylannja - rizni koljory

Posylannja HTML vidobrazaetjsja inshym koljorom zalezno vid togo, chy bulo vono vidvidane, ne vidviduvane chy actyvne

Koljory posylanj HTML

Za zamovchuvannjam posylannja vidobrazatymetjsja takym chynom (u vsih brauzerah):

  • Nevidvidane posylannja pidkreslene ta synje
  • Vidvidane posylannja pidkreslene i fioletove
  • Aktyvne posylannja pidkreslene i chervone

Vy mozete zminyty koljory stanu posylannja za dopomogoju CSS:
Pryklad
Tyt nevidvidane posylannja byde zelenym bez pidkreslennja. Vidvidane posylannja bude rozevym bez pidkreslennja. Actyvne posylannja bude zovtym i pidkreslenym.Krim togo,jakscho navesty kursor na posylannja (a:hover), vono stane chervonym i pidkreslenym:

style
a:link {
color: green;
background-color: transparent;
text-decoration: none;
}
a:visited {
color:pink;
background-color: transparent;
text-decoration: none;
}
a:hover{
color:red;
background-color: transparent;
text-decoration: underline;
}
a:active{
color:yellow;
background-color: transparent;
text-decoration: underline;
}
