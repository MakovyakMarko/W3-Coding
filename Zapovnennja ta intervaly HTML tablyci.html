Zapovnennja ta intervaly HTML tablyci

Tablyci html mozytj regyljuvaty zapovnennja vseredyni klitynok, a takoz prostir miz klitynkamy.

Tablycja HTML- Zapovnennja klitynok

Zapovnennja klitynok - ce prostir miz krajamy komirky ta vmistom komirky.
Za zamovchyvannjam zapovnennja vstanovleno na 0.
Schob dodaty vistyp do klitynok tablyci, vykorystovujte padding vlastyvistj CSS: Schob dodaty vidstup nad,pid,zliva i sprava vid zmistu vykorystovujte vlastyvosti padding-top padding-bottom padding-left padding-right

Tablycja HTML - interval miz klitynkamy

Vidstanj miz klitynkamy - ce prostir miz koznoju klitynkoju.
Za zamovchuvannjam probil vstanovlenyj na 2 pikseli.
Schob zminyty prostir miz klitynkamy tablyci, vykorystovujte border-spacing vlastuvistj CSS dlja table
