Element HTML picture

Element html picture nadae web-rozrobnykam biljshu gnuchkistj u vyznachenni resyrsiv zobrazenj.
Element picture mistytj odyn abo kiljka source elementiv, kozen z jakyh posylaetjsja na rizni zobrazennja cherez srcset atrybut. takym chynom brauzer moze vybrety zobrazennja, jake najkrasche vidpovidae potochnomu vygljadu ta/abo prystroju.
Kozen source element mae media atrybut, jakyj vyznachae, koly zobrazennja e najbiljsh pidhodjaschym.

Prymitka. Zavzdy vkazujte img element jak ostannij dochirnuj element picture elementa. Element img vykorystovuetjsja brayzeramy, jaki ne pidtrymujutj cej picture element, jakscho zoden z source tegiv ne vidpovidae.

Koly vykorystovuetjsja element zobrazennja

E dvi osnovni cili picture elementa:

1.Propuskna zdatnistj


Jakscho u vas malenjkyj ekran abo orystrij, ne potribno zavantazyvaty velykyj fail zobrazennja. Brayzer vykorystovuvatyme perschyj element iz vidpovidnym znachennjam atrybutiv i ignoruvatyme bydj-jakyj z nastupnyh elementiv.

2.Pidtrumka formatu

Dejaki brauzery abo prystrji mozutj pidtrymuvaty ne sci formaty zobrazenj. Vykorystovujuchy picture element, vy mozete dodavaty zobrazennja vsih formativ, i brauser vykorystovuvatyme pershyj format, jakyj vin rozpiznae, i ignoruvatyme budj-jakyj z nastupnyh elementiv.
