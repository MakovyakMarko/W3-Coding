Fonovi zobrazennja HTML

Fonove zobrazennja mozna vkazaty majze dlja budj-jakogo zobrazennja html.

Fonove zobrazennja na elementi HTML

Schob dodaty fonove zobrazennja do elementa html, vykorystovujte atrybut html i vlastyvistj style CSS: background-image

Vy takoz mozete vkazaty fonove zobrazennja v style elementi v head rozdili.
style
p{
backgpumd image: url('4.jpg');
}
/style

Fonove zobrazennja na storinci

Jakscho vy hochete, schob vsja storinka mala fonove zobrazennja, vu povynni vkazaty fonove zobrazennja dlja body elementa:
style
body {
background image: url ('4.jpg');
}

Fonovyj povtor

Jakscho fonove zobrazennja menshe elementa, zobrazennja povtorjuvatymatjsja po goryzontali ta vertykali, poky ne dosjagne kincja elementa.
Schob fonove zobrazennja ne povtorjuvalosj, vstanovitj dlja background-repeat vlastyvistj znachennja no-repeat.
style
body {
background-image: url('4.jpg');
background-repeat: no-repeat;
}

Fonova obkladynka

Jakscho vy hochete, schob fonove zobrazennja pokryvalo vesj element, vy mozete vstanovyty background-size vlastyvistj cover.
Krim togo, schob perekonatysja, scho vesj element zavzdy pokrytyj, vstanovitj background-attachment vlastyvistj fixed.
Takym chynom, fonove zobrazennja pokrye vesj element bez roztjaguvannja (zobrazennja zbereze svoji pochatkovi proporciji):
style
body {
background-image: url('4.jpg');
background-repeat: no-repeat;
background-attachment: fixed;
background-size: cover;
}
style

Roztjaguvannja fonu

Jakscho vy hochete, schob fonove zobrazennja roztjagnulosja na vesj element, vy mozete vstanovyty dlja background-size vlastyvistj 100% 100%. Z navedenyh vusche prykladiv vy diznalysj, scho fonovi zobrazennja mozna stylizuvaty za dopomogoju vlastyvostej fonu CSS.
