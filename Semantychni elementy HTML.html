Semantychni elementy HTML

Semantychni elementy = ce elemety zi znachennjam

Scho take semantychni elementy?

Semantychnyj element chitko opysue jogo znachennja jak dlja brauzera, tak i dlja rozrobnyka.
Pryklady nesemantychnyh elementiv: siv, span - nichogo ne rozpovidaje pro jogo zmist.
Pryklady semantychnyh elementiv: form, table, i article -chitko vyznachaje jogo zmist.

Semantychni elementy v HTML

Bagato web-sajtiv mistjatj HTML-kod, napryklad div id="nav", div class="header", div id="footer" dlja poznachennja navigaciji, verhnjogo ta nuzhnjogo kolontytuliv.
U html je dejaki semantychni elementy, jaki mozna vykorystvuvaty dlja vyznachennja riznyh chastyn web-storinky:

  • article
  • aside
  • details
  • figcaption
  • figure
  • footer
  • header
  • main
  • mark
  • nav
  • section
  • summary
  • time

HTML section element

Element section vyznachaje rozdil v dokumenti.
Vidpovidno do html-dokumentaciji w3c: "Rozdil - ce tematychne grupuvannja vmistu,jak pravylo, iz zagolovkom".
Pryklady, de section element mozna vykorystovuvaty:

  • Rozdily
  • Vstup
  • Novyny
  • Kontaktna informacija

Web-storinku zazvychaj mozna rozdilyty na rozdily dlja vstypy, vmisty ta kontaktnoji informaciji.

HTML article element

Element article vyznachje nezaleznyj, avtonomnyj vmist.
Stattja povunna maty sens sama po sobi, i maje buty mozlyvistj poshyrjuvaty jiji nezalezno vid reshty web-sajtu.
Pryklady, de element article mozna vykorystovuvaty:

  • Povidomlennja na forumi
  • Publikaciji v blozi
  • Komentari korystuvachiv
  • Fartky tovariv
  • Gazetni statti

Vkladennja article v section schy navpaky?

Element article vyznachaje avtonomnuj, nezaleznuj vmist.
Element section vyznachaje rozdil v dokumenti.
Chy mozemo my vykorystovuvaty vyznachennja, schob vyrishyty, jak vkladaty ci elementy? Ni, my ne mozemo!

Element HTML header

Element header javljaje soboju kontejner dlja vstupnodo vmistu, abo nabory navigacijnyh posylanj.
Zazvychaj header element mistytj:

  • Odyn abo kiljka elementiv zagolovkka (h1-h6)
  • Logotyp abo znachok
  • Vidomosti pro avtorstvo

Prymitka. V odnomu HTML-elemnti moze buty kiljka header elementiv. Odnak header jogo ne mozna pomistyty v footer, adress abo v inchyj header element.

Element html footer

Element footer vyznachaje nyznij kolontytul dlja dokumenta abo rozdilu.
Zazvychaj footer element mistytj:

  • vidomosti pro avtorstvo
  • informaciju pro avtorsjki prava
  • Kontaktna informacija
  • karta sajtu
  • povernutysja do verhnih posylanja
  • vidpovidni dokumenty

V odnomu dokumenti moze buty kiljka footer elementiv

HTML aside element

Element aside vyznachje dejakyj vmist, krim vmistu, v jakomu vin rozmischenyj (napryklad bichna panelj). Zmist aside maje buty oposeredkovano povjazanyj z navkolyshnim smistom.

Elementy HTML figure i figcaption

Element figure vyznachaje samostijnyj vmist, jak-ot iljustraciji,diagrany,fotografiji, spysky kodiv toscho.
Teg figcaption vyznachaje zagolovok dlja figure elementa. Element figcaption moze buty rozmischenyj jak pershyj abo jak ostannij dochirnij figure element.
