Elementy formatuvannja HTML

Elementy formatuvannja buly roztobleni dlja vidobrazennja specialjnuh typiv tekstu:

  • b - zurnuj tekst
  • strong - vazluvuj tekst
  • i - tekst kursyvom
  • em - Pidkreslenuj tekst
  • mark - poznachenuj tekst
  • small - menshyj tekst
  • del - vydalenyj tekst
  • ins - vstavlenyj tekst
  • sub - pidrjadkovyj tekst
  • sup - nadrjadkovyj tekst

Elementy html b i strong

This is bold
This is important!

Elementy html i i em

Element html i vyznachae chastynu tekstu aljternatyvnym golosom abo nastroem. Vmist vsereduni zazvuchaj vidobrazaetjsja kursyvom.
Teg i chasto vykorystovuetjsja dlja poznachennja tehnichnogo termina,frazy z inshoji movy,dumky, nazvy korablja toscho

HTML element del

Element html del vuznachae tekst, jakuj bulo vydaleno z dokumenta. Brauzery zazvychaj perebyrajutj rjadok cherez vydalenuj tekst:

My favorite color is blue red

Element HTML ins

Element html ins vyznachae tekst jakyj bulo vstavleno v document. Brauzery zazvychaj pidkresljujutj vstavlenuj tekst:

My favorite color is blue red

Element html sub

Element html sub vyznachae indeksnuj tekst. Pidrjadkovuj tekst zjavljaetsja na piv sumvola nuzche zvychajnogo rjadka i inodi vidobrazaetjsja menshym shryftom. Pidrjadkovyj tekst mozna vykorystovuvaty dlja himichnyh formyl H2O

This is subscript text

HTML element sup

Element sup vyznachae nadrjadkovyj tekst. Nadrjadkovyj tekst zjavljaetjsja na piv symvole vusche zvuchajnogo rjadka i inodi vidobrazaetjsja menshym shryftom. Nadrjadkovyj tekst mozna vykorystovuvaty dlja vynosok, napryklad WWW[1]
