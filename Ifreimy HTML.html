Ifrejmy HTML

HTML Iframe vykorystovuetjsja dlja vidobrazennja web-storinky na web-storinci.

Syntaksys HTML iframe

Teh HTML iframe vyznachaje cbudovanyj frejm.
Vbudovanyj frejm vykorystovuetjsja dlja vbydyvannja inshogo dokumenta v potochnuj HTML-dokument.
Syntaksys:
iframe src="url" title="description" /iframe
Porada. Rekomenduetjsja zavzdy vkljuchaty title atrybut dlja iframe. Ce vykorystovujetjsja programamy zchytuvannja z ekrana,schob zchytuvaty vmist iframe.

Pidsymok rozdilu

  • Teg html iframe vyznachaje vbydovanuj frame
  • Atrybut src vyznachaje URL-adresu storinky, jaku potribno vstavyty
  • Zavzdy vkljuchajte title atrybut (dlja program zchytuvannja z ekrana)
  • Atrybuty height ta width vyznachajutj rozmir iframe
  • Vykorystovujte border:none dlja vydalennja ramky navkolo iframe
  • Vykorystovujte name atrybut v iframe dlja posulannja na freim jak na ciljovyj kadr
