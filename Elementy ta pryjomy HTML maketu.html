    Elementy maketa HTML

    HTML maje kiljka semantychnyh elementiv, jaki vyznachajutj rizni chastyny web-storinky:

    • header- vyznachaje zagolovok dlja documenta abo rozdilu
    • nav - vyznachaje nabir navigacijnyh posylanj
    • section - vyznachaje rozdil u dokymenti
    • article - vyznachaje nezaleznyj, samostijnyj zmist
    • aside - vyznachaje vmist okremogo vmisty (napryklad bichna panelj)
    • footer - vyznachae nyznij kolontytul dlja dokymenta abo rizdilu
    • details- vyznachaje dodatkovi detali, jaki korystuvach moze vidkryvaty chy zakryvaty za vymogoju
    • summary - vyznachaje zagolovok dlja details elementa

    Metody rozmitky HTML

    Isnuje chotyry rizni tehniky stvorennja bagatokolonkovyh maketiv. Kozna tehnika maje svoji pljusy i minusy:

    • CSS framework
    • Vlastyvistj float CSS
    • CSS Flexbox
    • CSS sitka

    CSS Frameworks

    Jakscho vy hochete shvydko stvoryty svij maket, vy mozete vykorystovuvaty framework CSS, napryklad W3.CSS abo Bootstrap

    Maket CSS Float

    Mozna stvorjuvaty cili web-makety za dopomogoju float vlastyvosti CSS. Float legko navchytysja - vam prosto potribno zapamjataty, jak pracjujutj vlastyvosti float i clear. Nedoliky: Plavajuchi elementy pryvjazani do dokymentoobigu, scho moze zashkodyty gnuchkosti.

    Maket CSS Flexbox

    Vykorystannja flexbox garantuje, scho elementy vedutj sebe peredbachyvano, koly maket storinky maje vidpovidaty riznym rozmiram ekrana ta riznym prystrojam vidobrazennja.

    Maket sitky CSS

    Modylj CSS Grid Layuout proponuje systemu maketa na osnovi sitky z rjadkamy ta stovpcjamy, scho polegshuje rozrobku web-storinok bez vykorystannja float i pozycionuvannja.
