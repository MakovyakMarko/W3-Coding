Форми HTML

Форма HTML використовується для збору данних користувача. Введення користувача найчастіше надсилається на сервер для обробки.

<form action="/action_page.php">
<label for="fname">First name:</label><br/>
<input type="text" id="fname" name="fname" value="John"><br/>
<label for="lname">Last name:</label><br/>
<input type="text" id="lname" name="lname" value="Doe"><br/><br/>
<input type="submit" value="Submit">
</form>

Елемент <form>

Елемент HTML form використовується для створення форми HTML для введення користувача:

<form>
.
form elements
.
</form>

Елемент form є контейнером для різних типів елементів введення, таких як: текстові поля, прапорці, перемикач, кнопки надсилання тощо.

Елемент <input>

Елемент HTML input є найбільш використовуваним елементом форми.
Елемент input може відображатися багатьма способами, залежно від type атрибута.
Ось кілька прикладів:

Тип Опис
<input type="text"> Відображає однорядкове поле введення тексту
<input type="radio"> Відображає перемикач (для вибору одного з багатьох варіантів)
<input type="checkbox"> Відображає прапорець (для вибору нуля або більше з багатьох варінтів)
<input type="submit"> Відображає кнопку відправки (для подання форми)
<input type="button"> Відображає кнопку, яку можна натиснути

Текстові поля

Визначає <input type="text"> однорядкове поле для введення тексту.

Елемент <label>

Зверніть увану на використання label елемента в прикладі вище.
Тег label визначає мітку для багатьох елементів форми.
Елемент label корисний для користувасів програми зчитування з екрана, оскільки програма зчитування з екрпна вголос прочитає мітку, коли користувач зосередиться на елементі введення.
Елемент label також допомагає користувачам, яким важко клацнути на дуже маленьких ділянках (наприклад перемикачі або прапорці), оскільки, коли користувач натискає текст всередині label елементі він перемикає перемикач/прапорець.
Атрибут тега має бути рівним for атрибуту елемента, щоб зв'язати їх разом. <label> id <input>

Радіокнопки

Визначає <input type="radio"> перемикач.
Перемикач дозволяє користувачеві вибрати ОДИН з обмеженої кількості варіантів.

<form>
<input type="radio" id="html" name="fav_language" value="HTML">
<label for="html">HTML</label> <br>
<input type="radio" id="css" name="fav_language" value="CSS">
<label for="css">CSS</label> <br>
<input type="radio" id="javascript" name="fav_language" value="JavaScript">
<label for="javascript">JavaScript</label>
</form>

Прапорці

Визначає прапорець <input type="chekbox">
Пропорці дозволяють користувачеві вибрати нуль або більше параметрів з обмеженої кількості варіантів.

<form>
<input type="checkbox" id="vehicle1" name="vehicle1" value="Bike">
<label for="vehicle1">Я мене є велосипед</label>
<input type="checkbox" id="vihicle2" name="vehicle2" value="Car">
<label for="vehicle2">Я мене є машина</label>
<input type="checkbox" id="vehicle3" name="vehicle3" value="Boat">
<label for="vehicle3">В мене є човен</label>
</form>

Кнопка надіслати

Визначає кнопку надіслати <input type="submit"> кнопку для передачі даних форми обробнику форми.
Обробник форми зазвичай є файлом на сервері зі скриптом для обробки вхідних даних.
Обраобник форми вказується в action атрибуті форми.

<form action="/action_page.php">
<label for="fname">Ім'я:</label>
<input type="text" id="fname" name="fname" value="John">
<label for="lname">Фамілія:</label>
<input type="text" id="lname" name="lname" value="Doe">
<input type="submit" value="Відправити">
</form>

Атрибут імені для <input>

Зауважте, що кожне поле введення повинно мати name атрибут для посилання.
Якщо name атрибут опущено, значення поля взагалі не надсилатиметься.
