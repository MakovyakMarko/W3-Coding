Atrybut HTML id

Atrybut HTML id vykorystovuetjsja dlja vyznachennja unikaljnogo identyfikatora dlja elementa HTML.
U documenti HTML ne moze bytu biljshe odnogo elementa z odnakovym identyfikatorom.

Vykorystannja atrybuta id

Atrybut id vyznachaje ynikaljnyj isentyfikator dlja elementa HTML. Znachennja id atrybuta maje buty unikaljnym u HTML-documenti.
Atrybut id vykorystovuetjsja dlja vkazivky na pevne ogoloshennja stylju v tablyci styliv. Vin takoz vykorustovuetjsja JavaScript dlja dostupu ta keruvannja elementom iz pevnym identyfikatorom.
Syntaksys dlja id takuj: napyshitj hesh-symvol(#), a potim imja identyfikatora. Potim vyznachte vlastyvosti CSS u figyrnyh duzkah {}.
U nastypnomu prykladi mu majemo h1 element, jakyj vkazuje na imja identyfikatora "myHeader". Cej h1 element bude oformleno vidpovidno do #myHeader vyznachennja stylju v rozdili head:

My Header

Prymitka: imja identyfikatora chytlyve do registru!
Prymitka: imja identyfikatora maje mistyty prynajmni odyn symvol, ne moze pochynatusja z cyfry ta ne povynno mistyty probiliv(probiliv,tabuljaciji toscho).

Riznycja miz klasom i identyfikatorom

Imja klasy moze vkazuvatjsja kiljkoma elementamy HTML, todi jak imja identyfikatora maje vykorystovuvatysja lyshe odnym elementom HTML na storinci:

HTML-zakladky z identyfikatorom i posylannjamy

Zakladky html vykorystovujutjsja, schob dozvolyty chytacham perehodyty do pevnyh chastyn web-storinky.
Zakladky mozutj buty korysnymy, jakscho vasha storinka dyze dovga.
Schob vykorystaty zakladku, spochatku jiji potribno stvoryty, a potim dodaty posylannja na neji.
Potim, koly bude natysnuto posylannja, storinka byde prokruchuvatysja do miscja z zakladkoju.
Pryklad:
Spochatku stvoritj zakladku z id atrybutom:
h2 id=#C4 Chapter4
Potim dodajte posylannja na zakladku ("Perejty do rozdilu 4") z tijeji z storinky:
a href="#C4" jump to chapter4 /a
Abo dodajte posylannja na zakladku ("Perejty do rozdily 4") z inshoji storinky: a href="inshastorinka.html#C4" Jump to chapter 4 /a

Vykorystannja atrybuta id v JavaScript

Atrybut id takoz moze vykorystovuvatysja JavaScript dlja vykonannja dejakyh zavdanj dlja cjogo konkretnogo elementa.
JacaScript moze otrymaty dostyp do elementa z pevnym identyfikatorom za dopomogoju getElementById() metodu:
