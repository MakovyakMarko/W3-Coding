Spysok opysiv HTML

Spysok opysiv- ce spysok terminiv z opysom koznogo termina.
Teg dl vyznachaje spusok opysiv, teg dt vyznachae termin(nazvu), a teg dd opysuje kozen termin:

Coffee
-black hot drink
Milk
- white cold drink

Pidsumok rozdilu

  • Vykorystovujte HTML element dl dlja vyznachennja spysku opysiv
  • Vykorystovujte element HTML dt, schob vyznachutu termin opysu
  • Vykorystovujte element HTML dd, schob poysaty termin u spysku opysiv
