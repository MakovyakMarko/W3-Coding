CSS Margin Collapse

Іноді два поля згортаються в одне поле.

Згортання поля

Верхнє ти нижнє поля елементів іноді згортаються в одне поле, що дорівнює найбільшому з двох полів.

Цього не відбувається на лівому та правому полях! Лише верхні та нижні поля!

Подивіться на наступний приклад:

h1{
margin: 0 0 50px 0;
}
h2{
margin: 20px 0 0 0;
}

Відображатиметься так:

У наведеному вище прикладі елемент h1 має нижнє поле 50 пікселів, а елемент h2 має верхнє поле 20 пікселів.

Здоровий глузд припускає, що вертикальне поле між h1 i h2 становитиме 70 пікселів (50+20 пікселів). Але через згортання поля фактичне поле дорівнює 50 пікселям.
