Вертикальна панель навігації

Щоб побудувати вертикальну панель навігації, ви можете оформити елемент <a> всередині списку на додаток до коду з попередньої сторінки:

ul{
list-style-type:none;
margin: 0;
padding: 0;
}
li a{
display: block;
width: 60px;
background-color: #ddd;
}

Відоражатиметься так:

Пояснення.

  • display:block; - відображення посилань у вигляді блокових елементів робить клікабельною всю область посилання (а не лише текст), і це дозволяє нам вказати ширину (і відступи, поля, висоту тощо, якщо ви хочете)
  • width:60px; - елементи блоку займають повну ширину, доступну за замовчуванням, Ми хочемо вказати ширину 60пікселів

Ви також можете встановити ширину <ul> і видалити ширину <a>, оскільки вони займуть повну доступну ширину, коли відображатимуться як блокові елементи. Це дасть той самий результат, що й у попередньому прикладі

Приклад вертикальної панелі навігації

Створіть базову вертикальну панель навігації з сірим кольором фону та змініть колір фону посилань, коли користувач наводить на них мишу:

ul{
list-style-type: none;
margin: 0;
padding: 0;
width: 200px;
background-color: #f1f1f1;
}
li a{
display: block;
color: #000;
padding: 8px 16px;
text-decoration: none;
}
lia:hover{
background-color: #555;
color: white;
}

відображатиметься так:

Активне/поточне навігаційне посилання

Додайте клас "активний" до поточного посилання, щоб користувач знав, на якій сторінці він/вона:

.active{
background-color: #04aa6d;
color: white;
}

Відоражатиметься так:

Центруйте посилання та додайте межі

Додайте text-align:center до li або a, щоб цетрувати посилання.

Додайте border властивість до ul додайте раску навколо панелі навігоції. Якщо вам також потрібні межі всередині навігаційної панелі, додайте border-bottom до всіх елементів li, крім останнього:

ul{
border: 1px solid #555;
}
li{
text-align: center;
border-bottom: 1px solid #555;
}
li:last-child{
border-bottom: none;
}

Відоражатиметься так:

Повнорозмірна фіксована вертикальна панель навігації

Створіть "липку" бічну навігацію на повну висоту:

ul{
list-style-type: none;
margin: 0;
paddding: 0;
width: 25%;
background-color: #f1f1f1;
height: 100%;
position: fixed;
overflow: auto;
}

Відоражатиметься так:

Примітка. Цей приклад може не працювати належним чином на мобільних пристроях.
