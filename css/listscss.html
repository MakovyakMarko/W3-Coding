Списки HTML і властивості списку CSS

У HTML існує два основних типи списків:

  • невпорядковані списки (<ul>) - елементи списку позначені маркерами
  • впорядковані списки (<ol>) - елементи списку позначаються цифрами або літреами

Властивості списку CSS дозволяють:

  • Установити різні маркери елементів списку для впорядкованих списків
  • Установити різні маркери елементів списку для невпорядкованих списків
  • Установити зображення як маркер елемента списку
  • Додати фонові кольори до списків і елементів списку

Різні маркери елементів списку

Властивість list-style-type визначає тип маркера елемента списку.

У наступному прикладі показано деякі з доступних маркерів елементів списку:

ul.a{
list-style-type: circle;
}
ul.b{
list-style-type: square;
}
ol.c{
list-style-type: upper-roman;
}
ol.d{
list-style-type: lower-alpha;
}

Примітка. Деякі значення призначені для невпорядкованих списків, а деякі - для впорядкованих списків.

Зображення як маркер елемента списку

Властивість list-style-image визначає зображення як маркер елемента списку:

ul{
list-style-image: url('./img/1.png');
}

Розташуйте маркери елементів списку

Властивість list-style-position визначає положення маркерів елементів списку (маркованих точок).

"list-style-position: outside", означає, що маркери будуть розташовані за межами елемента списку. Початок кожного рядка елемента списку буде вирівняно по вертикалі. Це за умовчуванням.

"list-style-position: inside", означає, що маркери будуть усередині елемента списку. Оскільки він є частиною елемента списку, він буде частиною тексту та вставлятиме текст на початку.

ul.a{
list-style-position: outside;
}
ul.b{
list-style-position: inside;
}

Видалити налаштування за замовчуванням

Властивість list-style-type:none також можна використовувати для видалення маркерів/куль. Зверніть увагу, що список також має стандартні поля та відступи. Щоб видалити це, додайте margin:0 та padding:0 до ul або ol

ul{
list-style-type: none;
margin:0;
padding:0;
}

Список - коротка властивість

Властивість list-style є скороченою властивістю. Вона використовується для встановлення всіх властивостей списку в одні декларації:

ul{
list-style: circle inside url("./img/1.png");
}

При використанні скороченої властивості порядок значень властивості такий:

  • list-style-type (якщо вказано list-style-image, значення цієї властивості буде показано, якщо зображення з якоїсь причини не може бути відображено)
  • list-style-position (вказує, чи мають відображатися маркери елементів списку всередині чи поза потоком вмісту)
  • list-style-image (вказує заображення як маркер елемента списку)

Якщо одне з наведених вище значень властивості відсутнє, буде встановлено значення за умовчуванням для відсутньої властивості, якщо воно є.

Список стилів із кольорами

Ми також можемо стилізувати список за допомогою кольорів, щоб вони виглядали трохи цікавіше.

Усе, що додається до тегу <ol> або <ul>, впливає на весь список, тоді як властивості, додані до тегу <li>, впливатимуть на окремі елементи списку:

ol{
background: #ff9999;
padding: 20px;
}
ul{
background: #3399ff;
padding: 20px;
}
ol li{
background: #ffe5e5;
color: darker;
padding: 5px;
margin-left: 35px;
}
ul li{
background: #cce5ff;
color: darkblue;
margin: 5px;
}

border: 1px solid #ddd;
}

ul li {
padding: 8px 16px;
border-bottom: 1px solid #ddd;
}

ul li:last-child {
border-bottom: none;
}
