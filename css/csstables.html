Таблиці CSS

Вигляд таблиці HTML можна значно покращити за допомогою CSS:

#customers {
font-family: Arial, Helvetica, sans-serif;
border-collapse: collapse;
width: 100%;
}

#customers td, #customers th {
border: 1px solid #ddd;
padding: 8px;
}

#customers tr:nth-child(even){background-color: #f2f2f2;}

#customers tr:hover {background-color: #ddd;}

#customers th {
padding-top: 12px;
padding-bottom: 12px;
text-align: left;
background-color: #04AA6D;
color: white;
}

Межі таблиці

Щоб указати межі таблиці в CSS, використовуйте border властивість.

У наведеному нижче прикладі вказано суцільну рамку для елементів <table>,<th> i <td>:

table, th, td{
border: 1px solid;
}

Таблиця на повну ширину

У деяких випадках наведена вище таблиця може здатися маленькою. Якщо вам потрібна таблиця, яка має займати весь екран (на всю ширину), додайте width:100% до елемента <table>:

table{
width: 100%;
}

Відображатиметься так:

Подвійні межі: зверніть увагу, що таблиця в наведених вище прикладах має подвійні рамки.
Щоб видалити подвійні рамки, подивіться на приклад нижче.

Згорнути межі таблиці

Властивість border-collapse визначає, чи повинні межі таблиці згортатися в одну рамку:

teble{
border-collapse: collapse;
}

Відображатиметься так:

Якщо вам потрібна лише рамка навколо таблиці, вкажіть border властивість лише для <table>:

table{
border: 1px solid;
}
