Фони CSS

Властивості фону CSS використовуються для додавання фонових ефектів для елементів.
У цих розділах ви дізнаєтесь про такі властивості фону CSS:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background

Колір тла CSS

Властивість background-color визначає колір фону елемента.
Приклад. Колір фону сторінки встановлюється так:

body{
background-color: lightblue;
}

У CSS колір найчастіше визначається:

  • дійсна назва кольору, наприклад "червоний"
  • шістнадцядкове значення, наприклад "#ff0000"
  • значення RGB, наприклад "rgb(255,0,0)"

Інші елементи

Ви можете встановити колір фону для будь-яких елементів HTML:
Приклад. Тут елементи h1,p,div матимуть різні фонові кольори:

h1{
background-color: green;
}
div{
background-color: lightblue;
}
p{
background-color:yellow;
}

Непрозорість / Прозорість

Властивість opacity визначає непрозорість/прозорість елемента. Він може приймати значення від 0,0 до 1,0. Чим менше значення, тим прозоріше:

div{
background-color: green;
opacity: 0.3; }

Примітка. У разі використання opacity властивості для додавання прозорості фону елемента всі його дочірні елементи успадкують однакову прозорість. Це може ускладнити читання тексту всередині повністю прозорого елемента.

Прозорість за допомогою RGBA

Якщо ви не хочете застосувати непрозорість до дочірніх елементів, як у нашому прикладі вище, використовуйте значення кольорів RGBA. У наступному прикладі встановлюється непрозорість для кольору фону, а не таксту:

div{
background: rgba(0,128,0,0.3)
}
