Макет CSS - горизонтальне та вертикальне вирівнювання

Вирівнювання по центру елементів

Щоб горизонтально відцентрувати елемент блоку (наприклад, div), використовуйте margin: auto;

Встановленння ширини елемента запобігає його розтягуванню до країв контейнера.

Тоді елемент займе вказану ширину, а простір, що залишився, буде порівну розподілено між двома полями:

.center{
margin: auto;
width: 50%;
border: 3px solid green;
padding: 10px;
}

Примітка: вирівнювання по центру не має ефекту, якщо width властивість не встановлено (або встановлено на 100%).

Вирівнювання тексту по центру

Щоб просто відцентрувати текст всередині елемента, використовуйте text-align: center;

.center{
text-align: center;
border: 3px solid green;
}

Цетрування зображення

Щоб відцентрувати зображення, установіть ліве та праве поле auto ти зробіть його елементом block :

img{
display: block;
margin-left: auto;
margin-right: auto;
width: 40%;
}

Вирівнювання за лівим і правим краєм - використання position

Одним із методів вирівнювання елементів є використання position:absolute; :

.right{
position:absolute;
right: 0px;
width: 300px;
border: 3px solid #73AD21;
padding: 10px;
}

Відображатиметься так:

Примітка. Елементи з абосолютним розташуванням видаляються з нормального потоку та можуть перекривати елементи.

Вирівнювання по лівому та правому краю - за допомогою float

Іншим методом вирівнювання елементів є викиорстання float властивості:

.right{
float: right;
width: 300px;
border: 3px solid #73ad21;
padding: 10px;
}

По центру по вертикалі - за допомогою padding

У CSS є багато способів відцентрувати елемент вертикально. Просте рішення полягає в тому, що викоритсовувати top i bottom padding:

.center{
padding: 70px 0;
border: 3px solid green;
}

Відображатиметься так:

Щоб відцентрувати як вертикально, так і горизонтально, використовуйте padding та text-align:center:

.center{
padding: 70px 0;
border: 3px solid green;
text-align: center;
}

По центру по вертикалі - за допомогою line-height

Інший трюк полягає у використанні line-height властивості за значенням, рівним властивості height :

.center{
line-height: 200px;
height: 200px;
border: 3px solid green;
text-align: center;
}
Для багатострокового тексту введіть наступне:
.center p{
line-height: 1.5;
display: inline-block;
vertical-align: middle;
}

Центрувати по вертикалі - використання position i transform

Якщо padding i line-heigth не є варіантами, іншим рішенням є використання position та transform властивості:

.center{
height: 200px;
position: relative;
border: 3px solid green;
}
.center{
margin: 0;
position: absolute;
top: 50%;
left: 50%;
transform: translate (-50%, -50%);
}

По центру по вертикалі - за допомогою flexbox

Ви також можете використовувати flexbox для цетрування речей. Зверніть увагу, що flexbox не підтримується в IE10 і попередніх версіях:

.center{
display: flex;
justify-content: center;
align-items: center;
height: 200px;
border: 3px solid green;
}
