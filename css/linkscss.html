Посилання CSS

За допомогою CSS посилання можна стилізувати різними способами.

Стильні посилання

Посилання можна стилізувати за допомогою будь-якої властивості CSS (наприклад, color, font-family, background тощо.)

a{
color: hotpink;
}

Крім того, посилання можна оформляти по-різному залежно від того, в якому вони стані.

Чотири стани посилань:

  • a:link - звичайне, невідвідане посилання
  • a:visited - посилання, яке відвідав користувач
  • a:hover - посилання, коли користувач наводить на нього курсор
  • a:active - посилання в момент натискання
a:link{
color: red;
}
a:visited{
color: green;
}
a:hover{
color: hotpink;
}
a:active{
color: blue;
}

Відображатиметься так:

Примітка: при встановленні стилю для кількох станів посилання існують деякі правила порядку:

  • a:hover ПОВИНЕН бути після a:link i a:visited
  • a:active ПОВИНЕН бути після a:hover

Прикраса тексту

Властивість text-decoration переважно використовується для видалення підкреслень із посилань:

a:link{
text-decoration: none;
}
a:visited{
text-decoration: none;
}
a:hover{
text-decoration: underline;
}
a:active{
text-decoration: underline;
}

Колір фону

Властивість background-color можна використовувати для визначення кольору фону для посилань:

a:link{
background-color: yellow;
}
a:visited{
background-color: cyan;
}
a:hover{
background-color: lightgreen;
}
a:active{
background-color: hotpink;
}

Кнопки посилання

Цей приклад демонструє більш просунутий приклад, де ми об'єднуємо кілька властивостей CSS для відображення посилань у вигляді полів/кнопок:

a:link, a:visited{
background-color: #f44336;
color: white;
padding: 14px 25px;
text-align: center;
text-decoration: none;
display: inline-block;
}
a:hover, a:ative{
background-color: red;
}

Більше прикладів

Приклад. Цей приклад демонструє, як додати інші стилі до гіперпосилань

a.one:link{color:#ff0000;}
a.one.visited{color:#0000ff;}
a.one:hover{color:#ffcc00;}

a.two:link{color:#ff0000;}
a.two:visited{color:#0000ff;}
a.two:hover{font-size:150%;}

a.three:link{color:#ff0000;}
a.three:visited{color:#0000ff;}
a.three:hover{background:#66ff66;}

a.four:link{color:#ff0000;}
a.four:visited{color:#0000ff;}
a.four:hover{font-family:monospace;}

a.five:link{color:#ff0000; text-decoration:none;}
a.five:visited{color:#0000ff; text-decoration:none;}
a.five:hover{text-decoration:underline;}

Відображатиметься так:

Приклад. Інший приклад створення посилань/кнопок:

a:link,a:visited{
background-color: white;
color: black;
boder: 2px solid green;
padding: 10px 20px;
text-align: center;
text-decoration: none;
display:inline-block;
}
a:hover, a:active{
background-color: green;
color: white;
}
