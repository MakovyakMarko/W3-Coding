CSS налаштування висоти та ширини

Властивості height i width використовуються для встановлення висоти та ширини елемента.

Властивості висоти та ширини не включають відступи, раски чи поля. Вони встановлюють висоту/ширину області всередині відступу, межі та поля елемента.

Значення висоти та ширини CSS

Властивості height i width можуть мати такі значення:

  • auto - це стандартно. Браузер розраховує висоту і ширину
  • довжина - визначає висоту/ширину в px,cm тощо
  • % - визначає висоту/ширину у відсотках вмісту блоку
  • initial - встановлює значення за замовчуванням для висоти/ширини
  • inherit - висота/ширина буде успадкована від батьківського елемента

Приклад висоти та ширини CSS

div{
height: 200px;
width: 50%;
background-color: powderblue;
}

Відображатиметься так:

div{
height: 100px;
width: 500px;
background-color: powderblue;
}

Примітка. Пам'ятайте, що властивість height i width не містять відступів, рамок або полів! Вони встановлюють висоту/ширину області всередині відступу, межі та поля елемента!

Налаштування максимальної ширини

Властивість max-width використовується для встановлення максимальної ширини елемента.

Можна max-width вказати у значеннях довжини, як от px, cm тощо, або у відсотках (%) блоку, що містить елемет, або встановити значення "none"(це за замовчуванням. Це означає, що максимальної ширини немає).

Проблема з наведеним div вище виникає, коли вікно браузера менше за ширину елемента (500 пікселів). Потім браузер додає до сторінки горизонтальну смугу прокрутки.

Використання max-width натомість у цій ситуації покращить роботу браузера з малими вікнами.

Порада: перетягніть вікно веб переглядача на ширину 500 пікселів, щоб побачити різницю між двома елементами div! Примітка. Якщо ви з якоїсь причини використовуєте як width властивість, так і max-width властивість того самого елемента, і значення width властивості більше за max-width властивість - властивість max-width буде використано, а властивість width буде ігноруватись.

div{
max-width: 500px;
height: 100px;
/background-color: powderblue;
}
