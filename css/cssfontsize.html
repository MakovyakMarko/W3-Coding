Розмір шрифту CSS

Розмір шрифту

Властивість font-size втановлює розмір тексту.

У веб-дизайні важлива можливість керувати розміром тексту. Однак не слід використовувати коригування розміру шрифту, щоб обазци виглядали як заголовки, або заголовки виглядали як абзаци.

Завжди використовуйте відповідні теги HTML, наприклад <h1>-<h6> для заголовків і <p> для абзаців.

Значення розміру шрифту може бути абсолютним або відносним розіром.

Абсолютний розмір:

  • Встановлює заданий розмір тексту
  • Не дозволяє користувачеві змінювати розмрі тексту в усіх браузерах (погано з міркувань доступності)
  • Абсолютний розмір корисний, коли відомий фізичний розмір виведення

Відносний розмір:

  • Встановлює розмір відносно оточуючих елементів
  • Дозволяє користувачеві змінювати розмір тексту в браузерах

Примітка. Якщо ви не вкажете розмір шрифту, розмір за замовчуванням для звичайного тексту, наприклад абзаців, становить 16 пікселів (16 пікселів = 1em)

Установіть розмір шрифту з пікселями

Встановлення розміру тексту за допомогою пікселів дає вам повний контроль над розміром тексту:

h1{
font-size: 40px;
}
h2{
font-size: 30px;
}
p{
font-size: 14px;
}

Установіть розмір шрифту за допомогою Em

Щоб дозволити користувачам змінювати розмір тексту (в меню браузера), багато розробників використовують em замість пікселів.

1em дорівнює поточному розміру шрифту. Стандартний розмір тексту в браузерах становить 16 пікселів. Отже, стандартний розмір 1em становить 16 пікселів.

Розмір можна обчислити від пікселів до em за цією формулою: пікселі/16=em

h1{
font-size: 2.5em; (40px/16=2.5em)
}
h2{
font-size: 1.875em; (30px/16=1.875em)
}
p{
font-size: 0.875em; (14px/16=0.875em)
}

Відображатиметься так:

Використовуйте комбінацію відсотків і Em

Рішення, яке працює у всіх браузерах, полягає у встановленні стандартного розміру шрифту у відсотках для елемента <body>:

body{
font-size: 100%;
}
h1{
font-size: 2.5em;
}
h2{
font-size: 1.875em;
}
p{
font-size: 0.875em; }

Наш код тепер працює чудово! Він показує однаковий розмір тексту в усіх браузерах і дозволяє всім браузерам масштабувати або змінювати розмір тесту!

Чуйний розмір шрифту

Розмір тексту можна встановити за допомогою vw одиниць, що означає "ширину вікна перегляду" (viewport width)

Таким чином розмір тесту відповідатие розміру вікна браузера:

<h1 style="font-size: 10vw;">Responsove text</h1>
<p style="font-size: 5vw;">Resize the browser window to see how the text size scales.</p>
<p style="font-size: 5vw;">Use the "vw" unit when sizing the text. 10vw will set the size to 10% of the viewport width.</p>
