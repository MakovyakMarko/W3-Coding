Горизонтальна панель навігації

Існує два способи створити горизонтальну панель навігації. Використання вбудованих або плаваючих елементів списку.

Елементи вбудованого (inline) списку

Один зі способів створення горизонатльної навігаційної панелі - це вказати елементи li як вбудовані (inline) на додаток до стандортного коду з попередньої сторінки:

ul{
list-style-type: none;
margin: 0;
padding: 0;
}
li{
display: inline;
}

Відображатиметься так:

Пояснення.

  • display: inline - за умочуванням елементи li є блоковими. Тут ми видаляємо розриви рядків перед і після кожного елемента списку, щоб відобрадати їх в одному рядку

Елементи плаваючого списку

Інший спосіб створення горизонтальної навігаційної панелі полягає в тому, щоб перемістити елементи li та вказати макет для навігаційних посилань:

li{
float: left;
}
a{
display: block;
padding: 8px;
background-color: #ddd;
}

Відображатиметься так:

Пояснення.

  • float: left; - використовуйте float, щоб елементи блоку плавали один поруч з одним
  • display: block; - дозволяє нам вказати відступи (і висоту, ширину, поля тощо, якщо хочете)
  • padding: 8px; - вкажіть відступ між кожним елементом a, щоб вони виглядали добре
  • background-color: #ddd; - додайте сірий фоновий колір до кожного елемента a

Порада: додайте фоновий колір до ul замість кожного елемента а, якщо вам потрібен фоновий колір на всю ширину:

ul{
background-color: #ddd;
}

Приклади горизонтальної панелі навігації

Свторіть базову горищонтальну панель навігації з темним кольором фону та змініть колір фону посилань, коли коритсувач наводить на них мишу:

ul{
list-style-type: none;
margin:0;
padding: 0;
overflow: hidden;
background-color: #333;
}
li{
float: left;
}
li a{
display: block;
color: white;
text-align: center;
padding: 14px 16px;
text-decoration: none;
}
li a:hover{
background-color: #111;
}

Активне/поточне навігаційне посилання

Додайте клас "активний" до поточного посилання, щоб користувач знав, на якій сторінці він/вона:

.active {
background-color: #04aa6d;
}

Посилання за правим краєм

Вирівнюйте параграф за правим краєм, переміщуючи елементи списку праворуч (float: right):

<ul>
<li style="float:right;"><a class="active" href="#about"></li>
</ul>

Роздільники меж

Додайте border-right властивість до li, щоб створити роздільники посилань:

li{
border-ridght: 1px solid #bbb;
}
li:last-child{
border-right: none;
}
