Макет CSS - float i clear

Властивість CSS float визначає, як елемент має плавати.

Властивість CSS clear визначає, які елементи можуть плавати поруч із очищеним елементом і з якого боку.

Властивість float

Властивість float використовується для розміщеня та форматування вмісту, наприклад дозволити зображенню плавати ліворуч до тексту в контейнері.

Властивість float може мати одне з таких значень:

  • left - елемент плаває ліворуч від свого контейнера
  • right - елемент плаває праворуч від свого контейнера
  • none - елемент не плаває (відображатиметься лише там, де він зустрічається в тексті). Це за замовчуванням
  • inherit - елемент успадковує значення float свого батька

У своєму найпростішому використанні float властивість можна використовувати для обтікання текстом навколо зображення.

Приклад - float:right;

У наступному прикладі показано, що зображення має плавати праворуч у тексті:

img{
float: right;
}

Відображатиметься так:

Приклад float:left;

У наступному прикладі показано, що зображення має плавати ліворуч у тексті:

img{
float: left;
}

Відображатиметься так:

Приклад float:none;

У наступному прикладі зображення відображатиметься лише там, де воно зустрічається у тексті (float:none;):

img{
float: none;
}

Приклад - плавати поруч один з одним

Зазвичай елементи div відображаються один над одним. Однак, якщо ми використовуємо, float:left ми можемо дозволити елементам плавати поруч один з одним:

div{
float: left;
padding: 15px;
}
.div1{
background: red;
}
.div2{
background: yellow;
}
.div3{
background: green;
}
