CSS Google шрифти

Шрифти Google

Якщо ви не хочете використовувати жоден із стандартних шрифтів у HTML, ви можете скористатися Google Fonts.

Шрифти Google безкоштовні для використання та пропонують понад 1000 шрифтів на вибір.

Як користуватися Google Fonts

Просто додайте спеціальне посилання на таблицю стилів у розділі <head>, а потім зверніться до шрифту в CSS.

<head>
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Sofia">
<style>
body {
font-family:"Sofia", sans-serif;
}
</style>
</head>

Примітка. Указуючи шрифт у CSS, завжди вказуйте принаймні один резервний шрифт (щоб уникнути неочікуваною поведінки). Отже, також тут ви повинні додати загальне сімейство шрифтів (наприклад, serif або sans-serif) у кінець списку.

Використовуйте кілька шрифтів Google

Щоб використовувати декілька шрифтів Google, просто розділіть назви шрифтів вертикальною рискою ( "|" ), як тут:

<head>
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Audiowide|Sofia|Trirong">
<style>
h1.a {
font-family:"Audiowide",sans-serif;
}
h1.b{
font-family:"Sofia",sans-serif;
}
h1.c{
font-family:"Trirong", serif;
}
</style>
</head>

Відображатиметься так:

Примітка. Запит калькох шрифтів може сповільнити роботу веб-сторінок! Тому будьте обережні з цим.

Стилізуйте Google Fonts

Звичайно, ви можете стилізувати Google Fonts як завгодно за допомогою CSS!

<head>
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Sofia">
<style>
body {
font-family:"Sofia", sans-serif;
font-size: 30px;
text-shadow: 3px 3px 3px #ababab;
}
</style>
</head>

Увімкнення ефектів шрифту

Google також увімкнув різні ефекти шрифтів, які ви можете використовувати.

Спочатку додайте Google API, а потім додайте спеціальну назву класу до елемента, який використовуватиме спецефект. Ім'я класу завжди починається з font-efect і закінчується effectname.

<head>
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Sofia&effect=fire">
<style>
body {
font-family:"Sofia", sans-serif;
font-size:30px; }
</style>
</head>
<body>
<h1 class="font-effect-fire">Sofia on fire</h1>
</body>

Відображатиметься так:

Щоб запросити кілька ефектів шрифту, просто розділіть назви ефектів вертикальною рискою ("|"), як тут:

<head>
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Sofia&effect=neon|outline|emboss|shadow-multiple">
<style>
body {
font-family:"Sofia", sans-serif;
font-size:30px; }
</style>
</head>
<body>
<h1 class="font-effect-neon">Neon Effect</h1>
<h1 class="font-effect-outline">Outline Effect</h1>
<h1 class="font-effect-emboss">Emboss Effect</h1>
<h1 class="font-effect-shadow-multiple">Multiple Shadow Effect</h1>
</body>
