Коробкова модель CSS

Усі елементи HTML можна розглядати як коробки.

Модель CSS Box

У СSS термін "коробкова модель" використовується, коли йдеться про дизайн і макет.

Модель коробки CSS - це, по суті, коробка, яка обертається навколо кожного елемента HTML. Він складається з полів, рамок, відступів і фактичного вмісту. На зображення нижче показано коробкову модель:

Пояснення різних частин:

  • Вміст - вміст поля, де з'являються текст і зображення
  • Padding (відступ) - очищає область навколо вмісту. Відступ прозорий
  • Border (рамка) - межа, що охоплює відступи та вміст
  • Margin (поле) - очищає область за межами. Поле прозоре

Коробкова модель дозволяє додавати межі навколо елементів і визначати простір між елементами.

div{
width:300px;
border: 15px solid green;
padding: 50px;
margin: 20px;
}

Ширина і висота елемента

Щоб правильно встановити ширину і висоту елемента у всіх браузерах, потрібно знати, як працює модель коробки.

Важливо: коли ви встановлюєте властивості ширини та висоти елемента за допомогою CSS, ви просто встановлюєте ширину та висоту області вмісту. Щоб обчислити повний розмір елемента, ви також повинні додати відступи, межу та поля.

Приклад: Цей елемент div ьатиме загальну ширину 350 пікселів:

div{
width: 320px; padding: 10px; border: 5px solid grey; margin: 0; }

Ось розрахунок:
320 пікселів ширина;
+ 20 пікселів (правий та лівий відступ)
+ 10 пікселів (права та ліва межа)
+ 0 пікселів (праве та ліве поле)
= 350 пікселів

Загальну ширину елемента слід розраховувати так:

Загальна ширина елемента = ширина + лівий відступ + правий відступ + ліва межа + права межа + ліве поле + праве поле.

Загальну висоту елемента слів розраховувати так:

Загальна висота елемента = висота + верхній відступ + нижній відступ + верня межа + нижня межа + верхнє поле + нижнє поле.
