Псевдокласи CSS

Що таке псевдокласи?

Псевдоклас використовується для визначення особливого стану елемента.

Наприклад, його можна викорстовувати для:

  • Стилізації елемента, коли користувач наводить на нього мишу
  • По-різному стилізувати відвідані та невідвідані посилання
  • Стилізувати елемент, коли він отримає фокус

Синтаксис

Ситаксис псевдокласів:

selector:pseudo-class{
property: value;
}

Якірні псевдокласи

Посилання можуть відображатись різними способами:

unvisited link
a:link{
color: #ff0000;
}
a:visited{
color: #00ff00;
}
mouse over link
a:hover{
color: #ff00ff; }
selected link
a:active{
color: #0000ff;
}

Псевдокласи можна комбінувати з класами HTML:

Наведіть курсор на div

Приклад використання :hover псевдокласу в елементі div:

div:hover{
background-color: blue;
}

Просте наведення підказки

Наведіть курсор на елемент div, щоб показати елемент <p> (як підказка):

p{
display: none;
background-color: yellow;
padding: 20px;
}
div:hover p{
display: block;
}

CSS - псевдоклас :first-child

Псевдоклас :first-child відповідає зазначеному елементу, який є першим дочірнім елементом іншого елемента.

Зіставте перший елемент <p>

У наступному прикладі селектор відповідає будь-якому елементу <p>, який є першим дочірним елементом будь-якого елемента:

p:first-child{
color:blue;
}

Відображатиметься так:

Зіставте перший елемент <i> у всіх елементах <p>

У наступному прикладі селектор відповідає першому елементу <i> в усіх елементах <p> :

p i:first-child{
color: blue;
}

Відображатиметься так:

Зіставте всі елементи <i> у всіх перших дочірніх елементах <p>

У наступному прикладі селектор відповідає всім елементам <i> в елементах <p>, які є першими дочірніми елементами іншого елемента:

p:first-child i{
color: blue;
}

CSS - псевдоклас :lang

Псевдоклас :lang дозволяє визначити спаціальні правила для різних мов.

У наведеному нижче прикладі :lang визначає лапки для елементів <q> за допомогою lang="no":

<html>
<head>
<style>
q:lang(no){
quotes: "~" "~";
}
</style>
</head>
<body>
<p>Якийсь текст <q lang="no">Цитата в параграфі</q> Якийсь текст.</p>
</body>
</html>
