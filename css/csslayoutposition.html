Макет CSS - властивість position

Властивість position вказує тип методу позиціонування, який використовується для елемента (статичний, відносний, фіксований, абослютний або закріплений).

Властивості position

Властивість position визначає тип методу позиціонування, що використовується для елемента.

Є п'ять різних значень позиції:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Потім елементи розташовуються за допомогою властивостей top, bottom, left i right. Однак ці властивості не працюватимуть, якщо position властивість не встановлено спочатку. Вони також працюють по-різному залежно від значення позиції.

position:static;

Елементи HTML за замовчуванням позиціонуються статично.

На статичні позиціоновані елементи не впливають властивості top, bottom, left i right.

Елемент з position:static; не позиціонується ніяким особливим чином; він завжди розташовується відповідно до нормального потоку сторінки:

Цей елемент <div> має position:static

Ось CSS, який використовується:

div.static{
position: static;
border: 3px solid green;
}

position:relative;

Елемент position:relative; позиціонується відносно свого нормального положення.

Встановлення властивостей top, bottom, left, right віносно розташованого елемента призведе до його відхилення від нормального положення. Інший вміст не буде скориговано відповідно до проміжків, залишених елементом.

Цей елемент <div> має position:relative;

div.relative{
position: relative;
left: 30px;
border: 3px solid green;
}

position:fixed;

Елемент з position: fixed; позиціонується відносно вікна перегляду, що означає, що він завжди в тому самому місці, навіть якщо сторінку прокручують. Властивості top, right, bottom i left використовуються для позиціонування елемента.

Фіксований елемент не залишає розриву на сторінці, де він зазвичай розташовувався б.

Цей елемент <div> має position:fixed;.

Зверніть увагу на фіксований елемент у нижньому правому куті сторінки. Ось CSS, який використовується:

div.fixed{
position: fixed;
bottom: 0;
right:0;
width: 300px;
border: 3px solid green;
}

position:absolute;

Елемент з position: absolute; позиціонується відносно найближчого позиціонованого предка (замість позиціонування підносно вікна перегляду, як fixed)

Одак, якщо абслоютний позиціонований елемент не має позиціонування предків, він використовує тіло документа та переміщується разом із прокручуванням сторінки.

Примітка. Елементи з абослютним розташуванням видаляються з нормального потоку та можуть перекривати елементи.

Ось простий приклад:

Цей елемент <div> має position:relative;

Цей елемент <div> має position:absolute;

Ось CSS, який використовується:

div.relative{
position: relative;
width: 400px;
height: 200px;
border: 3px solid green;
}
div.absolute{
position: absolute;
top: 80px;
right: 0;
width: 200px;
height: 100px;
border: 3px solid green;
}

position:sticky;

Елемент з position:sticky; позиціонується на основі позиції прокручування користувача.

Закріплений елемент перемикається між relative i fixed залежно від положення прокручування. Він розташовується відносно, доки задане положення зсуву не зустрінеться у вікні перегляду - тоді воно "прилипає" на місце (наприклад, position:fixed;).

Примітка. Internet Explorer не підтримує фіксоване позиціонування. Для Safari потрібен префікс -webkit-. Ви також повинні вказати принаймні один з top,right,bottom,left, щоб закріплене позиціонування працювало.

У цьому прикладі липкий елемент прилопає до верхньої частини сторінки (top:0;), коли ви досягаєте його позиції прокручування.

div.sticky{
position: -webkit-sticky;
position: sticky;
top: 0;
background-color: green;
border: 2px solid #4CAF50;
}
