Округлені межі CSS

Властивість border-radius використовується для додавання округлених меж до елемента:

p.normal {
border: 2px solid red;
padding: 5px;
}

p.round1 {
border: 2px solid red;
border-radius: 5px;
padding: 5px;
}

p.round2 {
border: 2px solid red;
border-radius: 8px;
padding: 5px;
}

p.round3 {
border: 2px solid red;
border-radius: 12px;
padding: 5px;
}
