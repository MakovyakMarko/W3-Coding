Макет CSS - властивість z-index

Властивість z-index визначає порядок стека елемента.

Властивість z-index

Коли елементи розташовані, вони можуть перекривати інші елементи.

Властивість z-index визначає порядок стека елемента (який елемент має бути розміщений попереду чи позаду інших).

Елемент може мати позитивний або негативний порядок стека:

img{
position: absolute;
left: 0px;
top: 0px;
z-index: -1;
}

Примітка: z-index працює лише з позиціонованими елементами (position: absolute/relative/fixed/sticky) та flex елементами (елементами, які є прямими дочірніми елементами відображення: flex елементами)

Інший приклад z-index

Приклад. Тут ми бачимо, що елемент з більшим порядком стека завжди нод елементом з нижчим порядком стека:

.container {
position: relative;
}
.black-box{
position: relative;
z-index: 1;
border: 2px solid black;
height: 100px;
margin: 30px;
}
.gray-box{
position: absolute;
z-index: 3;
background: lightgray;
height: 60px;
width: 70%;
left: 50px;
top: 50px;
}
.green-box{
position:absolute;
z-index: 2;
background: lightgreen;
width: 35%;
left: 270px;
top: -15px;
height: 100px;
}

Без z-index

Якщо два позиціоновані елементи перекривають один одного без вказання z-index, елемент, визначений останнім у коді HTML, буде показано зверху.

.container {
position: relative;
}
.black-box{
position: relative;
border: 2px solid black;
height: 100px;
margin: 30px;
}
.gray-box{
position: absolute;
background: lightgray;
height: 60px;
width: 70%;
left: 50px;
top: 50px;
}
.green-box{
position:absolute;
background: lightgreen;
width: 35%;
left: 270px;
top: -15px;
height: 100px;
}
