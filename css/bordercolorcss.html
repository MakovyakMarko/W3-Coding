Колір рамки CSS

Властивість border-color використовується для встановлення кольору чотирьох рамок.

Колір можна встановити:

  • назва - вкажіть назву кольору, наприклад "червоний"
  • HEX - вкажіть значення HEX, наприклад "#ff0000"
  • RGB - вкажіть значення RGB, наприклад "rgb(255,0,0)"
  • HSL - вкажіть значення HSL, наприклад "hsl(0,100%,50%)"
  • прозорий

Примітка. Якщо border-color не встановлено, він успадковує колір елемента.

Приклад. Демонстрація різних кольорів рамки:

p.one {
border-style: solid;
border-color: red;
}

p.two {
border-style: solid;
border-color: green;
}

p.three {
border-style: dotted;
border-color: blue;
}

Відображатиметься так:

Red border

Green border

Blue border

Спеціальні кольори сторін

Властивість border-color може мати від одного до чотирьох значень (для верхньої межі, правої межі, нижньої межі та лівої межі).

p.four{
border-style: solid;
border-color: red green blue yellow;
}

Відображатиметься так:

A solid multycolor border

Шістнадцяткове значення

Колір рамки також можна вказати за допомогою шістнадцяткового значення (HEX):

p.five{
border-style: solid;
border-color: #ff0000;
}

Відображатиметься так:

A solid red border

Значення RGB

Або за допомогою значень RGB:

p.six {
border-style: solid;
border-color: rgb(255, 0, 0); /* red */
}

Відображатиметься так:

A solid red border

Значення HSL

Ви також можете використовувати значення HSL:

p.seven {
border-style: solid;
border-color: hsl(0, 100%, 50%);
}

Відображатиметься так:

A solid red border
