CSS коментарі

Коментарі CSS не відображаються в браузері, але вони можуть допомогти задокументувати вихідний код.

CSS коментарі

Коментарі використовуються для пояснення коду та можуть допомогти під час редагування вихідного коду пізніше.
Коментарі ігноруються браузерами.
Коментар CSS розміщується всередині style елемента та починається /* та закінчується на */:

/* This is single-line comment */
p {
color:red;
}

Ви можете додавати коментарі будь-де в коді:

p {
color:red; /* Set text color to red */
}

Коментарі також можуть займати кілька рядків:

/* This is
a multy-line
comment */
p {
color:red;
}
