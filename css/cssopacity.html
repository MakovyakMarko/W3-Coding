Непрозорість/прозорість CSS

Властивість opacity визначає напрозорість/прозорість елемента.

Прозоре зображення

Властивість opacity може приймати значення від 0,0 до 1,0. Чим менше значення, тим прозоріше:

img.a{
opacity:0.2;
}
img.b{
opacity:0.5;
}
img.c{
opacity:0.8;
}

Ефект прозорого наведення

Властивість opacity часто використовується разом із :hover селектором для зміни непрозорості при наведенні миші:

img{
opcity: 0.5;
}
img:hover{
opacity: 1.0;
}

Приклад з поясненням

Перший блок CSS схожий на код у прикладі 1. Крім того, ми додали, що має статися, коли користувач наводить курсор на одне із зображень. У цьому випадку ми хочемо, щоб зображення Не було прозорим. Коли користувач наводить на нього курсор воно стає прозорим:

img{
opacity: 1.0;
}
img:hover{
opacity: 0.5;
}

Прозора коробка

Під час використання opacity властивості для додавання прозорості фону елемента всі його дочірні елементи успадковують однакову прозорістью Це може ускладнити читання тексту всередині повністю прозорого елемента:

div{
background-color: #4caf50;
padding: 10px;
}
div.first{
opacity: 0.1;
}
div.second{
opacity: 0.3;
}
div.third{
opacity: 0.6;
}

Прозорість за допомогою RGBA

Якщо ви не хочете застосовувати до дочірніх елементів, як у нашому прикладі вище, використовуйте значення кольорів RGBA. У наступному прикладі встановлюється непрозорість для колору фону, а не тексту:

div{
background: rgb(76, 175, 80);
padding: 10px;
}
div.first{
background: rgba(76, 175, 80, 0.1);
}
div.second{
background: rgba(76, 175, 80, 0.3);
}
div.third{
background: rgba(76, 175, 80, 0.6);
}
