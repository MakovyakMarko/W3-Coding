Синтаксис CSS

Правило CSS складається з селектора та блоку оголошення.

Синтаксис CSS

Селектор вказує на елемент HTML, який потрібно стилізувати.
Блок декларацій містить одну або кілька декларацій, розділених крапкою з комою.
Кожна декларація містить назву властивості CSS і значення, роздалені двокрапкою.
Кілька декларацій CSS розділені крапкою з комою, а блоки декларацій оточені фігурними дужками.

p{
color:red;
text-align: center;
}

Відображатиметься так:

Пояснення прикладу:

  • p є селектором у CSS (він вказує на елемент HTML, який потрібно стилізувати: <p>)
  • color є властивістю та red є значенням властивості
  • text-align є властивістю та center є значенням властивості
