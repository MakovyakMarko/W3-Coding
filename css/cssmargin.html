CSS поля (margins)

Поля (margin) використовуються для створення простору навколо елементів за межами будь-яких визначених меж.

Поля СSS

Властивості CSS margin використовуються для створення простору навколо елементів поза будь-якими визначеними межами.

З СSS ви маєте повний контроль над полями. Існують властивості для встановлення поля для кожної сторони елемента (верхньої, правої, нижньої та лівої).

Поле - окремі сторони

CSS має властивості для визначення поля для кожної сторони елемента:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Усі властивості поля можуть мати такі значення:

  • auto - браузер розраховує поле.
  • довжина - вказує поле в px, pt,cm тощо.
  • % - визначає відступ у % від ширини елемента
  • inherit - вказує, що поле має бути успадковане від батьківського елемента

Порада: довзволені від'ємні значення.

p{
margin-top: 100px;
margin-botom: 100px;
margin-right: 150px;
margin-left: 80px;
}

Margin - скорочена властивість

Щоб скоротити код, можна вказати всі властивості поля в одній властивості.

Ця margin властивість є скоротченою властивістю для таких індивідуальних властивостей поля:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Отже, ось як це працює:

Якщо margin властивість має чотири значення:

  • поле: 25px 50px 75px 100px;
    • верхнє поле стиновить 25 пікселів
    • праве поле становить 50 пікселів
    • нижнє поле становить 75 пікселів
    • ліве поле становить 100 пікселів
p{
margin: 25px 50px 75px 100px;
}

Якщо margin властивість має три значення:

  • поле: 25px 50px 75px;
    • верхнє поле становить 25 пікселів
    • праве і ліве поле становить 50 пікселів
    • нижнє поле становить 75 пікселів
p{
margin: 25px 50px 75px;
}

Якщо margin властивість має два значення:

  • поле: 25px 50px;
    • верхнє і нижнє поле становлять 25 пікселів
    • праве і ліве поля становлять 50 пікселів
p{
margin: 25px 50px;
}

Якщо margin властивість має одне значення:

  • поле: 25px;
    • всі чотири поля становлять 25 пікселів
p{
margin: 25px;
}

Автоматичне значення

Ви можете встановити властивість margin auto на горизонтальне центрування елемента в його контейнері.

Тоді елемент займе вказану ширину, а простір, що залишився, буде порівну розподілено між лівим і правим полями.

div{
width: 300px;
margin: auto;
border: 1px solid red;
}

Успадковане значення

У цьому прикладі ліве поле елемента <p class="ex1"> успадковуєьтся від батьківського елемента (<div>):

div{
border: 1px solid red;
margin-left: 100px;
}
p.ex1 {
margin-left: inherit;
}
