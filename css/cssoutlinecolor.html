Колір контуру CSS

Властивість outline-color використовується для встановлення кольору контуру.

Колір можна встановити:

  • назва - вкажіть назву кольору, наприклад "red"
  • HEX - вкажіть шістнадцяткове значення, наприклад "#ff0000"
  • RGB - вкажіть значення RGB, наприклад "rgb(255,0,0)"
  • HSL - вкажіть значення HSL, наприклад "hsl(0,100%,50%)"
  • інвертувати - виконує інверсію кольорів (що забезпечує видимість контуру назелажно від кольору фону)

У наступному прикладі показано кілька різних контурів різними кольорами. Також зауважте, що ці елементи також мають тонку чорну рамку всередині контуру:

p.ex1{
border: 2px solid black;
outline-style: solid;
outline-color: red;
}
p.ex2{
border: 2px solid black;
outline-style: dotted;
outline-color: blue;
}
p.ex3{
border: 2px solid black;
outline-style: outset;
outline-color: grey;
}
