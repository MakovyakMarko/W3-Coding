Коротка властивість межі CSS

Границя CSS - властивість скорочення

Як ви бачили на попередній сторінці, існує багато властивостей, які слід враховувати при роботі з кордонами.

Щоб скоротити код, також можна вказати всі окремі властивості межі в одній властивості.

Властивість border є скоротченою властивіст для наступних окремих властивостей кордону:

  • border-width
  • border-style
  • border-color
p{
border: 5px solid red;
}

Ви також можете вказати всі окремі властивості межі лише для однієї сторони:

p{
border-left:6px solid red;
}

Відображатиметься так:

p{
border-bottom:6px solid red;
}
