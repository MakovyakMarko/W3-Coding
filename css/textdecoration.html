Оформлення тексту CSS

Прикраса тексту

У цьому розділі ви дізнаєтесь про такі властивості:

  • text-decoration-line
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness
  • text-decoration

Додайте декоративну лінію до тексту

Властивість text-decoration-line використовується для додавання декоративної лінії до тексту.

Порада. Ви можете комбінувати більше ніж одне значення, як-от підкреслення та підкреслення, щоб відображати рядки над текстом і під ним.

h1{
text-decoration-line: overline;
}
h2{
text-decoration-line: line-through;
}
h3{
text-decoration-line: underline;
}
p.ex{
text-decoration-line: overline underline;
}

Відображатиметься так:

Примітка. Не рекомендується підкреслювати текст, який не є посиланням, оскільки це часто заплутує читача.

Вкажіть колір для декоративної лінії

Властивість text-decoration-color використовується для встановлення кольору декоративної лінії.

h1{
text-decoration-line: overline;
text-decoration-color: red;
}
h2{
text-decoration-line: line-through;
text-decoration-color: blue;
}
h3{
text-decoration-line: underline;
text-decoration-color: green;
}
p.ex{
text-decoration-line: overline underline;
text-decoration-color: purple;
}

Вкажіть стиль для декоративної лінії

Властивість text-decoration-style використовується для встановлення стилю лінії декорування.

h1{
text-decoration-line: underline;
text-decoration-style: solid;
}
h2{
text-decoration-line: underline;
text-decoration-style: double;
}
h3{
text-decoration-line: underline;
text-decoration-style: dotted;
}
p.ex1{
text-decoration-line: underline;
text-decoration-style: dashed;
}
p.ex2{
text-decoration-line: underline;
text-decoration-style: wavy;
}
p.ex3{
text-decoration-line: underline;
text-decoration-color: red;
text-decoration-style: wavy;
}

Відображатиметься так:

Вкажіть товщину декоративної лінії

Властивість text-decoration-thickness використовується для встановлення товщини декоративної лінії.

h1{
text-decoration-line: underline;
text-decoration-thickness: auto;
}
h2{
text-decoration-line: underline;
text-decoration-thickness: 5px;
}
h3{
text-decoration-line: underline;
text-decoration-thickness: 25%;
}
p{
text-decoration-line: underline;
text-decoration-color: red;
text-decoration-style: double;
text-decoration-thickness: 5px;
}

Властивість скорочення

Властивість text-decoration є скороченою властивістю для:

  • text-decoration-line (вимагається)
  • text-decoration-color (не обов'язково)
  • text-decoration-style (не обов'язково)
  • text-decoration-thickness (не обов'язково)
h1{
text-decoration: underline;
}
h2{
text-decoration: underline red;
}
h3{
text-decoration: underline red double;
}
p{
text-decoration: underline red double 3px;
}

Невелика порада

Усі посилання в HTML за умовчанням підкреслені. Іноді ви бачите, що посилання оформлені без підкреслення. Використовується text-decoration: none; для видалення підкреслення з посилань, наприклад

a{
text-decoration: none;
}
