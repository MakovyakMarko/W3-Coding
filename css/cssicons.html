Іконки CSS

Піктограми можна легко додати до вашої HTML-сторінки за допомогою бібліотеки іконок.

Як додати значки

Найпростіший спосію додати піктограму на вашу HTML-сторінку - за допомогою бібліотеки піктограм, наприклад Font Awesome.

Додайте назву зазначеного класу піктограм до будь-якого вбудованого елемента HTML (наприклад, <i> або <span>).

Усі значки в наведених нижче бібліотеках значків є маштабованими векторами, які можна налаштовувати за допомогою CSS (розмір, колір, тінь тощо)

Значки шрифту Awesome

Щоб використовувати піктограми Font Awesome, перейдіть на fontawesome.com, увійдіть і отримайте код для додавання в <head> розділ вашої HTML-сторінки:

<script src="https://kit.fontawesome.com/yourcode.js" crossorigin="anonymous"></script>

Примітка: завантаження чи встановлення не потрібні!

<!DOCTYPE html>
<html>
<head>
<script src="https://kit.fontawesome.com/a076d05399.js" crossorigin="anonymous"></script>
</head>
<body>
<i class="fas fa-cloud"></i>
<i class="fas fa-heart"></i>
<i class="fas fa-car"></i>
<i class="fas fa-file"></i>
<i class="fas fa-bars"></i>
</body>
</html>

Піктограми Bootstrap

Щоб використовувати гліфіки Bootstrap, додайте такий рядок у <head> розділ вашої сторінки:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">

Примітка: завантаження чи встановлення не потрібні!

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css"></script>
</head>
<body>
<i class="glyphicon glyphicon-cloud"></i>
<i class="glyphicon glyphicon-remove"></i>
<i class="glyphicon glyphicon-user"></i>
<i class="glyphicon glyphicon-envelope"></i>
<i class="glyphicon glyphicon-thums-up"></i>
</body>
</html>

Іконки Google

Щоб використовувати піктограми Google, додайте такий рядок у <head> розділ вашої HTML-сторінки:

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Metarial+Icons">

Примітка: завантаження чи встановлення не потрібні!

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons"></script>
</head>
<body>
<i class="material-icons">cloud</i>
<i class="material-icons">favorite</i>
<i class="material-icons">attachment</i>
<i class="material-icons">computer</i>
<i class="material-icons">traffic</i>
</body>
</html>
