Фон CSS - властивість скоротчення

Щоб скоротити код, також можна вказати всі властивості фону в одній властивості. Це називається властивістю скорочення.
Замість написання:

body{
backgournd-color: #ffffff;
background-image: url("img/bg2.jpeg");
background-repeat: no-repeat;
background-position: right-top;
}

Ви можете використовувати властивість скорочення background:

body{
background: #ffffff url("img/bg2.jpeg") no-repeat right top; }

При використанні скороченної властивості порядок значень властивостей такий:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position;

Немає значення, якщо одне зі значень властивостей відсутнє, доки інші знаходяться в такому порядку. Зауважте, що ми не використовуємо властивість background-attachment у наведених вище прикладах, оскільки вона не має значення.
