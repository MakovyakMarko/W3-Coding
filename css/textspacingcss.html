Інтервал між текстом

У цьому розділі ви дізнаєтесь про такі властивості:

  • text-indent
  • letter-spacing
  • line-height
  • word-spacing
  • white-space

Відступи тексту

Властивість text-indent використовується для визначення відступу першого рядка тексту:

p{
text-indent: 50px;
}

Інтервал між літерами

Властивість letter-spacing використовується для визначення інтервалу між символами в тексті.

У наступному прикладі показано, як збільшити або зменшити пробіл між символами:

h2{
letter-spacing: 5px;
}
h3{
letter-spacing: -2px;
}

Висота лінії

Властивість line-height використовується для визначення інтервалу між рядками:

p.small{
line-height: 0.8;
}
p.big{
line-height: 1.8;
}

Інтервал між словами

Властивість word-spacing використовується для визначення інтервалу між словами в тексті.

У наступному прикладі показано, як збільшити або зменшити пробіл між словами:

p.one{
word-spacing: 10px; }
p.two{
word-spacing: -2px; }

Білий простір

Властивість white-space визначає, як обробляються пробіли всередині елемента.

Цей приклад демонструє, як вимкнути обтікання текстом всередині елемента:

p{
white-space: nowrap;
}
