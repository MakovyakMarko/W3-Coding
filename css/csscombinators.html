CSS комбінатори

Комбінатор - це щось, що пояснює звязок між селекторами.

Селектор CSS може містити більше одного простого селектора. Між простими селекторами ми миожемо включити комбінатор.

У CSS є чотири різні комбінатори:

  • селектор нащадка (пробіл)
  • дочірній селектор (>)
  • селектор сусіднього брата (+)
  • загальний селектор братів і сестер (~)

Вибір нащадка

Селектор нащадків відповідає всім елементам, які є нащадками зазначеного елемента.

У наступному прикладі вибираються всі елементи <p> всередині елементів <div>:

div p{
background-color: yellow;
}

Дочірній селектор (>)

Дочірній селектор вибирає всі елементи, які є дочірніми для зазначеного елемента.

div > p{
background-color: yellow;
}

Вибір сусіднього брата

Сусідній однорідний селектор використовується для вибору елемента, який знаходиться баспосередньо після іншого конкретного елемента.

Однорідні елементи повинні мати той самий батьківський елемент, а "суміжні" означає "відразу наступні".

У наступному прикладі вибирається перший елемент <p>, який розміщується відразу після елементів div:

div + p {
background-color: yellow;
}

Загальний вибір братів (~)

Загальний селектор братів вибирає всі елементи, які є наступними братами зазначеного елемента.

У наступному прикладі вибираються всі елементи <p>, які є наступними братами та сестрами елементів div

div ~ p {
background-color: yellow;
}
