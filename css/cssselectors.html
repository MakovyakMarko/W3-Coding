CSS селектори

Селектор CSS вибирає HTML-елементи, до яких потрібно додати стиль.

Селектори CSS

Селектори CSS використовуються для "знаходження" (або вибору) елементів HTML, які хочете стилізувати.
Ми можемо розділити селектори CSS на п'ять категорій:

  • Прості селектори (вибір елементів на основі імені, ідентифікатора, класу)
  • Селектори комбінаторів (вибір елементів на основі певного зв'язку між ними)
  • Селекотри псевдокласів (вибір елементів на основі певного стану)
  • Селектори псевдоелементів (вибір і стиль частини елемента)
  • Селектори атрибутів (вибір елементів на основі атрибутів або значення атрибута)

На цій сторінці пояснюються найпростіші селектори CSS.

Селектор елементів CSS

Селектор елементів вибирає елементи HTML на основі імені елемента.
Приклад. Тут усі елементи p на сторінці будуть вирівняні по центру з червоним кольором тексту:

p {
text-align:center;
color: red;
}

Селектор ідентифікатора CSS

Селектор id використовує атрибут id елемента HTML для вибору певного елемента.
Ідентифікатор елемента є унікальним на сторінці, тому селектор ідентифікатора використовується для вибору одного унікального елемента!
Щоб вибрати елемент із певним ідентифікатором, напишіть символ решітки ("#"), а потім ідентифікатор елемента.
Приклад. Наведене нижче правило CSS буде застосовано до елемента HTML з id="para1":

#para1{
text-align:center;
color: red;
}

Селектор класів CSS

Селектор класу вибирає елементи HTML із певним атрибутом класу.
Щоб вибрати елементи з певним класом, напишіть крапку ("."), а потім назву класу.
Приклад. У цьому прикладі всі елементи HTML із class="center" будуть червоними та вирівняні по центру:

.center{
text-align:center;
color: red;
}

Універсальний селектор CSS

Універсальний селектор ("*") вибирає всі елементи HTML на сторінці.
Приклад. Наведене нижче правило CSS впливає на кожен елемент HTML на сторінці

*{
text-align:center;
color: blue;
}

Селектор групування CSS

Селектор групування вибирає всі елементи HTML з однаковими визначеннями стилю.
Подивіться на наступний код CSS (елементи h1,h2,p мають однакові визначення стилю):

h1 {
text-align:center;
color: red;
}
h2 {
text-align:center;
color: red;
}
p {
text-align:center;
color: red;
}

Краще буде згрупувати селектори, щоб згорнути код.
Щоб згрупувати селектори, розділіть їх комою.
Приклад. У цьому прикладі ми згрупували селектори з коду вище:

h1,h2,p {
text-align:center;
color: red;
}
