Стиль рамки CSS

Властивість border-style визначає тип рамки для відображення.

Допускаються такі значення:

  • dotted - Визначає пунктирну (крапками) рамку
  • dashed - Визначає пунктирну рамку
  • solid - Визначає суцілну рамку
  • double - Визначає подвійну рамку
  • groove - Визначає 3D рифлену межу. Ефект залежить від значення кольору рамки
  • ridge- Визначає 3D-ребристу межу. Ефект залежить від значення кольору рамки
  • inset- Визначає рамку 3D-вставки. Ефект залежить від значення кольору рамки
  • outset- Визначає початкову межу 3D. Ефект залежить від значення кольору рамки
  • none- Не визначає межі
  • hidden- Визначає приховану межу

Властивість border-style може мати від одного до чотирьох значень (для верхньої мержі, правої межі, нижньої межі, та лівої межі)

p.dotted border-style: dotted;
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}

Примітка. Жодна з ІНШИХ властивостей кордону CSS (про які ви дізнаєтесь більше в наступних розділах) не матиме ЖОДНОГО ефекту, якщо border-style властивість не встановлено!
