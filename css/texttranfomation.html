Транформація тексту

Властивість text-transform використовується для визначення великих і малих літер у тексті.

Його можна використовувати, щоб перетворити все на великі чи малі літери або зробити першу літеру кожного слова великою:

p.uppercase{
text-transform: uppercase;
}
p.lowercase{
text-transform: lowercase;
}
p.capitalize{
text-transform: capitalize;
}
