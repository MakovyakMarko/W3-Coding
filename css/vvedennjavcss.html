Введення в HTML

CSS - це мова, яку ми використовуємо для оформлення веб-сторінки.

Що таке CSS?

  • CSS розшифровується як каскадні таблиці стилів
  • CSS описує, як елементи HTML мають відображатися на екрані, папері чи в інших місцях
  • CSS економить багато роботи. Він може керувати макетом кількох веб-сторінок одночасно
  • Зовнішні таблиці стилів зберігаються у файлах CSS

Навіщо використовувати CSS?

CSS використовується для визначення стилів ваших веб-сторінок, включаючи дизайн, макет і варіації відображення для різних пристроїв і розмірів екрана.

body {
background-color: lightblue;
}
h1 {
color: white;
text-align: center;
}
p {
font-family: verdana;
font-size: 20px;
}

CSS вирішив велику проблему

HTML ніколи не мав на меті містити теги для форматування веб-сторінки!
HTML був створений для опису вмісту веб-сторінки, наприклад:
<h1>Це заголовок</h1>
<p>Це абзац.</p>
Коли до специфікації HTML 3.2 було додано такі теги як <font> і атрибути кольору, для веб розробників почався кошмар. Розробка великих веб-сайтів, де до кожної окремої сторінки додавалися шрифти та інформація про колір, стала тривалим і дорогим процесом.
Щоб вирішити цю проблему, Консорціум Всесвітньої павутини (W3C) створив CSS.
CSS видалив форматування стилю зі сторінки HTML!

CSS економить багато роботи!

Визначення стилів зазвичай зберігаються у зовнішніх файлах .css.
За допомогою зовнішнього файлу таблиці стилві ви можете змінити вигляд цілого веб-сайту, змінивши лише один файл!
