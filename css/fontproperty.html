Властивість шрифту CSS

Щоб скоротити код, також можна вказати всі окремі властивості шрифту в одній властивості.

Властивість font є скороченою властивістю для:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

Примітка. Значення font-size та font-family є обов'язковим. Якщо одне з інших відсутнє, використовується значення за умовчанням.

p.a{
font: 20px Arial,sans-serif;
}
p.b{
font: italic small-caps bold 12px/30px Georgia, serif;
}
