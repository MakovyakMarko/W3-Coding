Стиль таблиці CSS

Простір таблиці

Щоб контролювати простір між межею та вмістом у таблиці, використовуйте padding властивість елементів <td> i <th>:

th,td{
padding: 15px;
text-align:left;
}

Горизонтальні роздільники

Додайте border-bottom властивість до <th> i <td> для горизонтальних роздільників

th,td{
border-bottom: 1px solid #ddd;
}

Таблиця з наведенням

Використовуйте :hover селектор на <tr>, що виділити рядки таблиці при наведенні миші:

tr:hover{background-color: coral;}

Смугасті таблиці

Для таблиць зі смужками зебри використовуйте nth-child() селектор ш додайте background-color до всіх парних (або непарних) рядків таблиці:

tr:nth-child(even) {background-color: #f2f2f2;}

Колір таблиці

У наведеному нижче прикладі вказано колір фону та колір тексту елементів <th>:

th{
background-color:#04aa6d;
color: white;
}
