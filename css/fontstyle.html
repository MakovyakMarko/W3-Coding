Стиль шрифту CSS

Стиль шрифту

Властивість font-style здебільшого використовується для визначення тексту курсивом.

Ця властивість має три значення:

  • normal - текст відображається нормально
  • italic - текст відображається курсивом
  • oblique - текст "нахилений (oblique дуже схожий на курсив, але менше підтримується)
p.normal{
font-style: normal;
}
p.italic{
font-style: italic;
}
p.oblique{
font-style: oblique;
}

Варіант шрифту

Властивість font-variant визначає, чи повинен текст відображатись малим шрифтом.

У невеликому шрифті всі малі літери перетворюються на великі. Однак перетворені великі літери відображаються меншим розміром шрифту, ніж вихідні великі літери в тексті.

p.normal{
font-variant: normal;
}
p.small{
font-variant: small-caps;
}
