СSS Padding

Властивість CSS padding використовується для створення простору вмісту елемента всередині будь-яких визначених меж.

З CSS ви маєте повний контроль над відступами. Існують властивості для встановлення відступів для кожної сторони елемента (верхньої, правої, нижньої та лівої).

Padding - окремі сторони

CSS має властивості для визначення відступу для кожної сторони елемента:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Усі властивості padding можуть мати такі значення:

  • довжина - визначає відступ у px, pt, cm тощо.
  • % - визначає відступ у % від ширини елемента
  • inherit - вказує, що заповнення має бути успадковане від батьківського елемента

Приимітка: від'ємні значення не допускаються.

div{
padding-top: 50px;
padding-right: 30px;
padding-bottom: 50px;
padding-left: 80px;
}

Padding - властивість скоротчення

Щоб скоротити код, можна вказати всі властивості заповнення в одній властивості.

Властивість padding є скоротченою властивістю для наступних окремих властивостей заповнення:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Отже, ось як це працює:

Якщо padding властивість має чотири значення:

  • padding: 25px 50px 75px 100px;
    • верхній відступ становить 25 пікселів
    • правий відступ становить 50 пікселів
    • нижній відступ становить 75 пікселів
    • лівий відступ становить 100 пікселів

Якщо padding властивість має три значення:

  • padding: 25px 50px 75px
    • верхній відступ становить 25 пікселів
    • правий і лівий відступи становлять 50 пікселів
    • нижній відступ становить 75 пікселів

Якщо padding властивість має два значення:

  • padding: 25px 50px;
    • верхній і нижній віступи становлять 25 пікселів
    • правий і лівий відступи становлять 50 пікселів

Якщо padding властивість має одне значення:

  • padding: 25px;
    • усі чотири відступи мають 25 пікселів
div.four{
padding: 25px 50px 75px 100px;
}
div.three{
padding: 25px 50px 75px;
}
div.two{
padding: 25px 50px;
}
div.one{
padding: 25px;
}

Відступ і ширина елемента

Властивість CSS width визначає ширину області вмісту елемента. Область вмісту - це частина всередині заповнення, рамки та поля елемента (модель коробки (box model)).

Отже, якщо елемент має задану ширину, відступ, доданий до цього елемента, буде додано до загальної ширини елемента. Це часто є небажаним результатом.

Приклад. Тут елементу div надається ширина 300 пікселів. Однак фактична ширина елемента становитиме 250 пікселів (300 пікселів + 25 пікселів зліва + 25 пікселів справа):

div{
width:300px;
padding: 25px;
}

Щоб зберегти ширину на рівні 300 пікселів, незалежно від кількості відступів, ви можете використовувати box-sizing властивість. Це змушує елемент зберігати свою фактичну ширину; якщо ви збільшите відступ, доступний простір вмісту зменшиться.

div{
width:300px;
padding: 25px;
box-sizing: border-box;
}
