Макет CSS - властивість відображення

Властивість display є найважливішою властивістю CSS для керування макетом.

Властивість дисплея

Властивість display визначає, чи/як буде відображатися елемент.

Кожен елемент HTML має значення відображення за замовчуванням залежно від типу елемента. Стандартним значення відображення для більшості елементів є block або inline.

Елементи рівня блоку

Елемент рівня блоку завжди починається з нового рядка і займає всю доступну ширину (розтягується ліворуч і праворуч, наскільки це можливо).

Елемент <div> є елементом рівня блоку

Приклади елементів рівня блоку:

  • <div>
  • <h1>-<h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>

Вбудовані елементи

Вбудований елемент не починається з нового рядка і займає лише необхідну ширину.

Це вбудований елемент <span> всередині абзацу

Приклади вбудованих елементів:

  • <span>
  • <a>
  • <img>

display: none

display: none; зазвичай використовується з JavaScript, щоб приховати та показати елементи без їх видалення та повторного створення. Подивіться на наш останній приклад на цій сторінці, якщо ви хочете знати, як цього можна досягти.

Елемент <script> використовує display:none; за замовчуванням.

Замінити відображуване значення за замовчуванням

Як згадувалося, кожен елемент має значення відображення за замовчуванням. Однак ви можете перевизначити це.

Зміна вбудованого елемента на блоковий елемент або навпаки може бути корисною для надання сторінки певного вигляду та дотримання веб-стандартів.

Типовим прикладом є створення вбадованих <li> елементів для горизонтального меню:

li{
display: inline;
}

Примітка. Налаштування властивості відображення елемента змінює лише спосіб відображення елемента, а НЕ те, який це елемент. Таким чином, вдудованому (inline) елементу з display: block; заборонено мати інші блокові елементи всередині нього.

У наступному прикладі елементи <span> відображаються як блокові елементи:

span{
display: block;
}

Приховати елемент - display:none або visibility:hidden

Приховати елемент можна, встановивши для display властивості значення none. Елемент буде приховано, а сторінка відображатиметься так, ніби елемента там немає:

<head>
<style>
h1.hidden {
display: none;
}
</style>
</head>
<body>

<h1>This is a visible heading</h1>
<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the h1 element with display: none; does not take up any space.</p>

</body>

Відображатиметься так:

visibility:hidden; також приховує елемент.

Однак елемент займатиме стількиж місця, скільки й раніше. Елемент буде приховано, але це все одно вплине на макет:

<head>
<style>
h1.hidden {
visibility: hidden;
}
</style>
</head>
<body>

<h1>This is a visible heading</h1>
<h1 class="hidden">This is a hidden heading</h1>
<p>Notice that the hidden heading still takes up space.</p>

</body>
