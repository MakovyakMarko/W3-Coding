CSS адаптивна таблиця

Чуйна таблиця

Адаптивна таблиця відображатиме горизонтальну смугу прокрутки, якщо екран замалий для відображення всього вмісту.

Додайте елемент контейнера (наприклад, <div>) overflow-x:auto навколо елемента <table>, щоб зробити його адаптивним:

<div style="overflow-x:auto;">
<table>
... table content ...
</table>
</div>
