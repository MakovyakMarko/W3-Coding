Макет CSS - приклади float

Ця сторінка містить типові приклади float.

Сітка коробок / коробки однакової ширини

За допомогою float властивості можна розміщувати блоки вмісту поруч:

*{
box-sizing: border-box;
}
.box{
float: left;
width: 33.33%;
padding: 50px;
}
.clearfix::after{
content: "";
clear: both;
display: table;
}

Відображатиметься так:

Що таке box-sizing?
Ви можете легко створити три плаваючі коробки поруч. Однак, коли ви додаєте щось, що збільшує ширину кожного поля (наприклад відступ від межі), поле розривається. Властивість box-sizing дозволяє нам включити відступ і рамку в загальну ширину (і висоту) коробки, переконавшись, що відступ залишається всередині коробки і не ламається.

Зображення поруч

Сітку прямокутників також можна використовувати для відображення зображень поруч:

*{
box-sizing: border-box;
}
.img-conteiner{
float: left;
width: 33.33%;
padding: 5px;
}
.clearfix::after{
content: "";
clear: both;
display: table;
}

Коробки однакової висоти

У попередньму прикладі ви навчилися розміщувати коробки поруч із однаковою шириною. Однак створоити плаваючі короблки однакової висоти нелегко. Однак швидким виправленням є встановлення фіксованої висоти, як у прикладі нижче:

*{
box-sizing: border-box;
}
.box{
float:left;
width:50%;
padding:50px;
height: 300px;
}
.clearfix::after{
content: "";
clear: both;
display: table;
}

Відображатиметься так:

Однак це не дуже гнучко. Це нормально, якщо ви можете гарантувати, що в ящиках буде однакова кількість вмісту. Але часто зміст не однаковий. Якщо ви спробуєте наведений вище приклад на мобільному телефоні, ви побачите, що вміст другої коробки відображається за її межами. Ось тут і стане в нагоді CSS3 Flexbox, оскільки він може автоматично розтягувати коробки до найдовшої коробки:

.flex-container{
display: flex;
flex-wrap: nowrap;
background-color: DodgerBlue;
}
.flex-container .box{ background-color: #f1f1f1;
width: 50%;
margin: 10px;
text-align: center;
line-height: 75px;
font-size: 30px;
}

Навігаційне меню

Ви також можете використовувати float зі списком гіперпосилань, щоб створити горизонтальне меню:

ul{
list-style-type: none;
margin:0;
padding:0;
overflow:hidden;
background-color:#333;
}
li{
float:left;
}
li a{
display: inline-block;
color: white;
text-align: center;
padding: 14px 16px;
text-decoration: none;
}
li a:hover{
background-color: #111;
}
.active{
background-color: red;
}

Приклад веб-макету

Також часто можна створювати цілі веб-макети за допомогою float властивості:

*{
box-sizing: border-box;
}
body{
background-color: white;
}
.header, .footer{
background-color: grey;
color: white;
padding: 15px;
}
.column{
float: left;
padding: 15px;
}
.clearfix::after{
content: "";
clear: both;
display: table;
}
.menu{
width: 25%;
}
.content{
width: 75%;
}
.menu ul{
list-style-type: none;
margin: 0;
padding: 0;
}
.menu li{ padding: 8px;
margin-bottom: 8px;
background-color: #33b5e5;
color: #ffffff;
}
.menu li:hover{
background-color: #0099cc;
}

Відображатиметься так:

Більше прикладів

Нехай перша літера плаває ліворуч:

span{
float: left;
width: 0.7em;
font-size: 400%;
font-family: algerian, courier;
line-height: 80%;
}
