Псевдоелементи CSS

Що таке псевдоелементи?

Псевдоелементи CSS використовуються для стилізації визначених частин елемента.

Наприклад, його можна використовувати для:

  • Стилю першої літери або рядка елемента
  • Додавання вмісту до або після вмісту елемента

Синтаксис

Синтаксис псевдоелементів:

selector::pseudo-element{
property: value;
}

Псевдоелемент ::first-line

Псевдоелемент ::first-line використовується для додавання особливого стилю до першого рядка тексту.

У наступному прикладі форматується перший рядок тексту в усіх елементах <p>:

p::first-line{
color: #ff0000;
font-variant: small-caps;
}

Примітка. Псевдоелемент ::first-line можна застосувати лише до елементів рівня блоку.

Наступні властивості застосовуються до ::first-line псевдоелемента:

  • font
  • color
  • background
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-hight
  • clear

Псевдоелемент ::first-letter

Псевдоелемент ::first-letter використовується для додання особливого стилю першій літері тексту.

У наступному прикладі форматується перша літера тексту в усіх елементак <p>:

p::first-letter{
color: #ff0000;
font-size: xx-large;
}

Відображатиметься так:

Примітка. Псевдоелемент ::first-letter можна застосовувати лише до елементів рівня блоку.

Наступні властивості застосовуються до псевдоелемента ::first-letter:

  • font
  • color
  • background
  • margin
  • padding
  • border
  • text-decoration
  • vertical-align (only if "float" is "none")
  • text-transform
  • line-height
  • float
  • clear

Псевдоелементи та класи HTML

Псевдоелементи можна комбінувати з класами HTML:

p.intro::first-letter{
color: red;
font-size: 200%;
}

Кілька псевдоелементів

Кілька псевдоелементів також можна комбінувати

У наступному прикладі перша літера абзацу буде червоного кольору, шрифт розміром хх. Інша частина першого рядка буде синьою і написана великими літерами. Решта абзацу буде стандартним розміром і кольором шрифту:

p::first-letter{
color: #ff0000;
font-size: xx-large;
}
p::first-line{
color: #0000ff;
font-variant: small-caps;
}

CSS - псевдоелемент ::before

Псевдоелемент ::before можна використовувати для вставки деякого вмісту перед вмістом елемента.

У наступному прикладі зображення встановлюється перед вмістом кожного елемента <h1>:

h1::before{
content: url(./img/smile.jpeg);
width: 30px;
height: 30px;
}

CSS - псевдоелемент ::after

Псевдоелемент ::after можна використовувати для вставки деякого вмісту після вмісту елемента.

У наступному прикладі зображення вставляється після кожного елемента <h1>:

h1::after{

CSS - псевдоелемент ::marker

Псевдоелемент ::after вибирає маркери елементів списку.

Наступний приклад стилізує маркери елементів списку:

::marker{
color: red;
font-size: 23px;

CSS псевдоелемент ::selection

Псевдоелемент ::selection відповідає частині елемента, яку вибирає користувач.

Наступні властивості CSS можна застосувати до ::selection : color, background, cursor, outline.

У наступному прикладі виділений текст стає червоним на жовтому тлі:

::selection{
color: red;
background: yellow;
}
