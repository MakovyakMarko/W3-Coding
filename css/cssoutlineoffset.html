CSS Outline Offset

Властивість outline-offset додає простір між контуром і межею елемента. Простір між елементом і його контуром прозорий.

У наступному прикладі вказано контур на 15 пікселів за межами краю.

p{
margni:30px;
border: 1px solid black;
outline: 1px solid red;
outline-offset: 15px;
}

p{
margni:30px;
background: yellow;
border: 1px solid black;
