Макет CSS - переповнення (overflow)

Властивість CSS overflow контролює, що відбувається з вмістом, який занадто великий, щоб поміститися в область.

Переповнення CSS

Властивість overflow визначає, чи потрібно обрізати вміст або додати смуги прокручування, якщо вміст елемента занадто великий, щоб поміститися у вказану область.

Властивість overflow має такі значення:

  • visible - за замовчуванням. Переповнення не обрізається. Вміст відображається за межами вікна елемента
  • hidden - переповнення буде обрізано, а решта вмісту буде невидимою
  • scroll - переповнення обрізано, а для перегляду решти вмісту додано смугу прокрутки
  • auto - подібно до scroll, але смуги прокрутки додаються лиша за потреби

Примітка: властивість overflow працює лише для блокових елементів із заданою висотою.

Примітка. В OS X Lion (на Mac) смуги прокрутки приховані за умовчанням і відображаються лише під час використання (навіть якщо встановлено "overflow:scroll").

overflow:visible

За замовчуванням переповнення має visible значення, що означає, що воно не обрізається та відображається за межами вікна елемента:

div{
width: 200px;
height: 65px;
background-color: coral;
overflow: visible;
}

Відображатиметься так:

overflow:hidden

За допомогою hidden значення переповнення обрізається, а решта вмісту прихована:

div{
overflow:hidden;
}

Відображатиметься так:

overflow:scroll

Якщо встановити значення scroll, переповнення буде обрізано та додано смугу прокрутки для прокручування всередині поля. Зауважте, що це додасть смугу прокрутки як горизонтально, так і вертикально (навіть якщо вона вам не потрібна):

div{
overflow: scroll;
}

Відображатиметься так:

overflow:auto

Значення auto подібне до scroll, але воно додає смуги прокрутки лише за необхідності:

div{
overflow: auto;
}

Відображатиметься так:

overflow-x i overflow-y

Властивість overflow-x і overflow-y визначають, чи потрібно змінювати переповнення вмісту лише горизонтально чи вертикально (або обидва):

div{
overflow-x: hidden;
overflow-y: scroll;
}
