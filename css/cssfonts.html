Шрифти CSS

Важливо вибрати правильний шрифт для вашого сайту!

Вибір шрифту важливий

Вибір правильного шрифту має величезний вплив на сприйняття веб-сайту читачими.

Правильний шрифт може створити ідентичність вашого бренду.

Важливо використовувати шрифт, який легко читається. Шрифт додає цінності тексту. Також важливо вибрати правильний колір і розмір тексту для шрифту.

Загальні сімейства шрифтів

У CSS існує п'ять загальних сімейств шрифтів:

  1. Шрифти з засічками мають невелику риску по краяз кожної літери. Вони створюють відчуття офіційності та елегантності.
  2. Шрифти без засічок мають чіткі лінії. Вони створюють сучасний і мінімалістичний вигляд.
  3. Моноширинні шрифти - тут усі літери мають однакову фіксовану ширину. Вони створюють механічний вигляд.
  4. Курсивні шрифти імітують людьський почерк.
  5. Фентезійні шрифти - це декоративні/грайливі шрифти.

Усі різні назви шрифтів належать до однієї із загальних сімейств шрифтів.

Різниця між шрифтами Serif i Sana-Serif

Примітка. На екранах комп'ютерів шрифти без засічок легше читати, ніж шрифти з засічками.

Властивість сімейства шрифтів CSS

У CSS ми використовуємо font-family властивість для визначення шрифту тексту.

Примітка. Якщо назнва шрифту складається з кількох слів, вона має бути взята в лапки, наприклад "Times New Roman".

Порада: властивість font-family має містити кілька назв шрифтів як "запасну" систему, щоб забезпечити максимальну сумісність між браузерами/операційними системами. Почніть із шрифту, який вам потрібен, і закінчіть загальним сімейством (щоб дозволити браузеру вибрати подібний шрифт із загального сімейства, якщо інші шрифти недоступні). Назви ширфтів слід розділяти комами.

.p1{
font-family: "Times New Roman", Times, serif;
}
.p2{ font-family: Arial, Helvetica, sans-serif;
}
.p3{
font-family: "Lucida Console", "Courier New", monospace;
}
