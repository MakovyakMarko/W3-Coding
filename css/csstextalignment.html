Вирівнювання тексту CSS

Вирівнювання тексту та напрямок тексту

У цьому розділі ви дізнаєтесь про такі властивості:

  • text-align
  • text-align-last
  • direction
  • unicode-bidi
  • vertical-align

Вирівнювання тексту

Властивість text-align використовується для встановлення горизонтального вирівнювання тексту.

Текст можна вирівняти за лівим або правим краєм, по центру або за шириною

У наведеному нижче прикладі показано вирівнювання тексту по центру, ліворуч і праворуч (вирівнювання за лівим краєм за замовчуванням, якщо напрямок тексту зліва направо, і вирівнювання за правим краєм за замовчуванням, якщо напрямок тексту справа наліво):

h1{
text-align: center;
}
h2 {
text-align: left;
}
h3{
text-align: right;
}

Коли для text-align властивості встановлено значення "justify", кожен рядок розтягується так, щоб кожен рядок мав однакову ширину, а ліве та праве поля були прямими (як у журналах і газетах)

div{
text-align: justify;
}

Вирівнювання останнього рядка

Властивість text-align-last визначає спосіб вирівнювання останнього рядка тексту.

p.a{
text-align-last: right; }
p.b{
text-align-last: center;
}
p.c{
text-align-last: justify;
}

p{
direction: rtl;

Вертикальне вирівнювання

Властивість vertical-align задає вертикальне вирівнювання елемента.

img.a{
vertical-align: baseline;
}
img.b{
vertical-align: text-top;
}
img.c{
vertical-align: text-bottom;
}
img.d{
vertical-align: sub;
}
img.e{
vertical-align: super;
}
