Вирівнювання таблиці CSS

Горизонтальне вирівнювання

Властивість text-align встановлює горизонтальне вирівнювання (наприклад, ліворуч, праворуч або по центру) вмісту в <th> або <td>.

За замовчуванням вміст елементів <th> вирівнюється по центру, а вміст елементів <td> вирівнюється за лівим краєм.

Щоб також вирівняти вміст елементів <td> по центру, використовуйте text-align:center :

td{
text-align: center;
}

Щоб вирівняти вміст за лівим краєм, примусово вирівнюйте елементи <th> за лівим краєм за допомогою text-align:left властивості:

th{
text-align: left;

Вертикальне вирівнювання

Властивість vertical-align встановлює вертикальне вирівнювання (наприклад, зверху, знизу, або посередині) вмісту в <th> або <td>.

За замовчуванням вміст у таблиці вирівнюється посередині (для елементів <th> i <td>).

У наступному прикладі встановлюється вертикальне вирівнювання тексту за низом для елементів <td>:

td{
height: 50px:
vertical-align: bottom;
}
