Безпечні веб-шрифти CSS

Що таке безпечні веб-шрифти?

Безпечні веб-шрифти - це шрифти, які універсально встановлюються на всіх браузерах і пристроях.

Резервні шрифти

Однак не існує 100% повністю базпечних веб-шрифтів. Завжди існує ймовірність того, що шрифт не знайдено або встановлено неправильно.

Тому дуже важливо завжди використовувати резервні шрифти.

Це означає, що вам слід додати список подібних "резервних шрифтів" у font-family властивості. Якщо перший шрифт не працює, браузер спробує наступний і так далі. Завжди закінчуйте список загальною назвою сімейства шрифтів.

Приклад. Тут є три типи шрифтів: Tahoma, Verdana та sans-serif. Другий і третій шрифти є резервними, на випадок, якщо перший не знайдено

p{
font-family: Tahoma, Verdana, sans-serif;
}

Найкращі безпечні веб-шрифти для HTML i CSS

Нижче наведено список найкращих безпечних веб-шрифтів для HTML i CSS:

  • Arial (без засічок)
  • Verdana (без засічок)
  • Tahoma (без засічок)
  • Trebuchet MS (без засічок)
  • Times New Roman (з засічками)
  • Georgia (з засічками)
  • Garamond (з засічками)
  • Courier New (моноширинний)
  • Brush Script MT (курсив)

Примітка. Перш ніж публікувати свій веб-сайт, завжди перевіряйте, як ваші шрифти виглядають у різних браузерах і пристроях, і завжди використовуйте резервні шрифти!

Arial (без засічок)

Arial є найпоширенішим шрифтом як для онлайн, так і для друкованих ЗМІ. Arial також є шрифтом за замовчанням у Google Documents

Arial є одним із найбезпечніших веб-шрифтів, і він доступний у всіх основних операційних системах.

body{
font-family: Arial, sans-serif;
}

Verdana (без засічок)

Verdana - дуже популярний шрифт. Verdana легко читається навіть за малих розмірів шрифту.

body{
font-family: Verdana, sans-serif;
}

Tahoma (без засічок)

Шрифт Tahoma має менше місця між символами.

body{
font-family: Tahoma, sans-serif;
}

Відображатиметься так:

Trebuchet MS (без засічок)

Trebuchet MS розроблено Microsoft у 1996 році. Використовуйте цей шрифт обережно. Підтримується не всіма мобільними операційними системами.

body{
font-family: 'Trebuchet MS', sans-serif;
}

Відображатиметься так:

Times New Roman (з засічками)

Times New Roman - один із найвідоміших шрифтів у світі. Це виглядає професійно та використовується в багатьох газетах і на "новинних" веб-сайтах. Це також основний шрифт для пристроїв і програм Windows.

body{
font-family: 'Times New Roman', serif;
}

Відображатиметься так:

Georgia (з засічками)

Georgia - елегантний шрифт із засічками. Його добре читати з різними розмірами шрифту, тому він є хорошим кандидатом для дизайну, адаптованого до мобільних пристроїв.

body{
font-family: Georgia, serif;
}

Відображатиметься так:

Garamond (з засічками)

Garamond - це класичний шрифт, який використовується для багатьох друкованих книг. Він має пазачасовий вигляд і добре читається.

body{
font-family: Garamond, serif;
}

Відображатиметься так:

Courier New (моноширинний)

Courier New - це найпоширеніший моноширинний шрифт із засічками. Courier New часто використовується кодовими дисплеями, і багато постачальників послуг електронної пошти використовують його як стандартний шрифт. Courier New також є стандартним шрифтом для кіносценаріїв.

body{
font-family: 'Courier New', monospace;
}

Відображатиметься так:

Brush Script MT (курсив)

Шрифт Brush Script MT створено для імітації рукописного введення. Він елегантний і вишуканий, але його важко читати. Використовуйте його обережно.

body{
font-family: 'Brush Script MT', coursive;
}
