Кольори CSS

Кольори вказуються за допомогою попередньо визначених назв кольорів або значень RGB, HEX, HSL, RGBA, HSLA

Назви кольорів CSS

У СSS колір можна вказати за допомогою попередньо визначеної назви кольору:

Tomato

Orange

DodgerBlue

MediumSeaGreen

Gray

SlateBlue

Violet

LightGray

Колір фону CSS

Ви можете встановити колір фону для елементів HTML:

<h1 style="background-color:DodgerBlue;">Hello World</h1>
<p style="background-color:Tomato;">Lorem ipsum dolor sit, amet consectetur adipisicing elit. Deserunt molestias enim nam quasi eaque fugiat, error dolor eligendi odio explicabo quibusdam eveniet, quod sunt est!</p>

Відображатиметься так:

Hello World

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Beatae tempore ab iure expedita, aliquid consequatur nesciunt eveniet culpa maxime consequuntur nisi enim quasi quibusdam recusandae.

Колір тексту CSS

Ви можете встановити колір тексту:

<h1 style="color:Tomato;">Hello World</h1>
<p style="color:DodgerBlue;">Lorem ipsum...</p>
<p style="color:MediumSeaGreen;">Ut wisi enim...</p>

Відображатиметься так:

Hello World

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Officiis facilis dolorem, optio laudantium deleniti quas.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat neque eligendi qui a, earum nihil.

Колір рамки CSS

Ви можете встановити колір рамок:

<h1 style="border:2px solid Tomato;">Hello World</h1>
<h1 style="border:2px solid DodgerBlue;">Hello World</h1>
<h1 style="border:2px solid Violet;">Hello World</h1>

Відображатиметься так:

Hello World!

Hello World!

Hello World!

Значення кольорів CSS

У СSS кольори також можна вказати за допомогою значень RGB,HEX,HSL,RGBA та HSLA:
Те саме, що назва кольору "Tomato":

<h1 style="background-color:rgb(255, 99, 71);">...</h1>
<h1 style="background-color:#ff6347;">...</h1>
<h1 style="background-color:hsl(9, 100%, 64%);">...</h1>
