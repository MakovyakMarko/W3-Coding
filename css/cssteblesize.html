Розмір таблиці CSS

Ширина та висота таблиці

Ширина та висота таблиці визначаються властивостями width i height.

Наведений нижче приклад встановлює ширину таблиці на 100%, а висоту елементів th на 70 пікселів:

table{
width: 100%;
}
th{
height: 70px;
}

Відображатиметься так:

Щоб створити таблицю, яка має займати лише половину сторінки, використовуйте width:50% :

table{
width:50%;
}
