Вкладення фону CSS (background-attachment)

CSS фонове вкладення

Властивість background-attachment визначає, чи має фонове зображення прокручуватись чи бути фіксованим (не прокручуватись разом з рештою сторінки):
Приклад. Вкажіть, що фонове зображення має бути фіксоване:

body{
background-image: url("img/bg2.jpeg");
whether background-repeat: no-repeat;
background-position: right top;
background-attachment: fixed;
}

Приклад. Вкажіть, що фонове зображення має прокручуватись разом із рештою сторінки:

body{
background-image: url("img/bg2.jpeg");
background-repeat: no-repeat;
background-position: right top;
background attachment: scroll;
}
