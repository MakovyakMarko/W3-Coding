Ширина межі CSS (border-width)

Властивість border-width визначає ширину чотирьох меж.

Ширину можна встановити як певний розмір (у px,pt,cm,em тощо) або за допомогою одного з трьох попередньо визначених значень: тонкий, середній, або товстий:

p.one{
border-style: solid;
border-width: thin;
}
p.two{
border-style: solid;
border-width: medium;
}
p.three{
border-style: solid;
border-width: thick;
}
p.four{
border-style: solid;
border-width: 10px;
}

Конкретні ширини сторін

Властивість border-width може мати від одного до чотирьох значень (для верньої межі, правої межі, нижньої межі, лівої межі):

p.five {
border-style: solid;
border-width: 5px 20px; /* 5px top and bottom, 20px on the sides */
}

p.six {
border-style: solid;
border-width: 20px 5px; /* 20px top and bottom, 5px on the sides */
}

p.seven {
border-style: solid;
border-width: 25px 10px 4px 35px; /* 25px top, 10px right, 4px bottom and 35px left */
}
