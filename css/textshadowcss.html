Властивість text-shadow додає тінь до тексту

У найпростішому випадку ви вказуєте лише горизонтальну тінь (2px) і вертикальну тінь (2px):

h1{
text-shadow: 2px 2px;
}

Відображатиметься так:

Далі додайте колір (червоний) до тіні:

h1{
text-shadow: 2px 2px red;
}

Відображатиметься так:

Потім додайте до тіні ефект розмиття (5px):

h1{
text-shadow: 2px 2px 5px red;
}

Відображатиметься так:

Більше прикладів тіні тексту:

Тінь тексту на більму тексті:

h1{
color: white;
text-shadow: 2px 2px 4px #000000;
}

Відображатиметься так:

Текст-тінь з червоним неоновим світінням:

h1{
text-shadow: 0 0 3px #ff0000;
}

Відображатиметься так:

Текст-тінь з червоним і синім неоновим світінням:

h1{
text-shadow: 0 0 3px #ff0000, 0 0 5px #0000ff;
}

Відображатиметься так:

Ше один приклад тіні

h1{
color:white;
text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;
}
