Фонове зображення CSS

Властивість background-image визначає зображення, яке буде використовуватись як фон елемента.
За замовчуванням зображення повторюється, тому воно покриває весь елемент.
Приклад. Установіть фонове зображення для сторінки:

body{
background-image: url("img/gradient_wp.jpeg");
}

Примітка. Використовуючи фонове зображення, використовуйте зображення, яке не заваджає тексту.
Фонове зображення також можна встановити для певних елементів, наприклад елемента <p>:

p{
background-image: url("img/gradient_wp.jpeg")
}
