Чудові пари шрифтів CSS

Гарне поєдняння шрифтів є важливим для чудового дизайну.

Правила поєднання шрифтів

Ось кілька основних правил створення чудових пар шрифтів:

1. Доповнення

Завжди безпечно знайти пари шрифтів, які доповнюють один одного.

Чудове поєднання шрифтів має гармоніювати, не бути надто схожими чи надто різними.

2. Використовуйте суперсімейства шрифтів

Суперсімайства шрифтів - це набір шрифтів, створениз для ефективної взаємодії. Таким чином використання різних шрифтів в одному суперсімействі є безпечним.

Наприлад, суперсімейство Lucida містить такі шрифти: Lucida Sans, Lucida Serif? Lucide Typewriter Sans, Lucida Typewriter Serif i Lucida Math.

3. Контраст є королем

Два занадто сзожі шрифти часто конфліктують. Однак правильно зроблені контрасти виявляють найкраще в кожному шрифті.

Приклад: поєднання засічок із засічками є добре відомим поєднанням.

Потужне суперсімейство включає варіанти одного шрифту як із засічками, так і без засічок (наприклад, Lucida та Lucida Sans)

4. Вибкріть лише одного боса

Один шрифт повинен бути головним. Це встановлює ієрархію для шрифтів на вашій сторінці. Цього можна досягти, змінюючи розмір, жирність та колір.

body {
font-family: Verdana, sans-serif;
font-size: 16px;
color: gray;
}
h1{
font-family: Goergia, serif;
font-size: 60px;
color: black;
}
