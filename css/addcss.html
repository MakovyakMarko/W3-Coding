Як додати CSS

Коли браузер читає таблицю стилів, віе форматує документ HTML відповідно до інформації в таблиці стилів.

Три способи вставити CSS

Існує три способи вставлення таблиці стилів:

  • Зовнішній CSS
  • Внутрішній CSS
  • Вбудований CSS

Зовнішній CSS

За допомогою зовнішньої таблиці стилів ви можете змінити вигляд цілого веб-сайту, змінивши лише один файл!
Кожна HTML-сторінка має містити посилання на зовнішній файл таблиці стилів всередині елемента <link> у розділі head.
Приклад. Зовнішні стилі визначають в елементі link у розділі head сторінки HTML:

<!DOCTYPE html>
<html lang="en">
<head>
<link rel="stylesheet" href="mystyle.css">
</head>
<body>
<h1>This is heading</h1>
<p>This is paragraph</p>
</body>
</html>

Зовнішню таблицю стилів можна написати в будь-якому текстовому редакторі, її потрібно зберегти з розширенням .css
Зовнішній файл .css не повинен містити тегів HTML.
Ось як виглядає файл "mystyle.css":

body{
background-color:lightblue;
}
h1 {
color:navy;
margin-left:20px;
}

Відображатиметься так:

Примітка. Не додавайте пробіл між значенням властивості та одиницею:
Неправильно(пробіл): margin-left: 20 px;
Правильно(без пробілу): margin-left: 20px;

Внутрішній CSS

Внутрішню таблицю стилів можна використовувати, якщо одна сторінка HTML має унікальний стиль.
Внутрішній стиль визначається всередині елемента style всередині head.
Приклад. Внутрішні стилі визначаються в елементі style всередині розділу head сторінки HTML:

<!DOCTYPE html>
<html lang="en">
<head>
<style>
body{
background-color: linen;
}
h1{
color: maroon;
margin-left: 40px;
}
</style>
</head>
<body>
<h1>This is heading</h1>
<p>This is paragraph</p>
</body>
</html>

Вбудований CSS

Вбудований стиль можна використовувати для застосування унікального стилю до окремого елемента.
Щоб використовувати вбудовані стилі, додайте атрибут style до відповідного елемента. Атрибут style може містити будь-яку властивість CSS.
Приклад. Вбудовані стилі визначаються в атрибуті style відповідного елемента:

<!DOCTYPE html>
<html lang="en">
<head>
</head>
<body>
<h1 style="color:blue; text-align:center;">This is heading</h1>
<p style="color:red;">This is paragraph</p>
</body>
</html>

Відображатиметься так:

Порада: вбудований стиль втрачає багато переваг таблиць стилів (через змішування вмісту з представленням). Використовуйте цей метод помірно.

Кілька таблиць стилів

Якщо деякі властивості були визначені для того самого селектора (елемента) в різних таблицях стилів, буде використано значення з останньої прочитаної таблиці стилів.
Припустімо, що зовнішня таблиця стилів має такий стиль для елемента <h1>:

h1{
color:navy;
}

Потім припустимо, що внутрішня таблиця стилів також має такий стиль для елемента <h1>:

h1{
color:orange;
}

Приклад. Якщо внутрішній стиль визначено після посилання на зовнішню таблицю стилів, елементи <h1> будуть "orange"(помаранчевими):

<!DOCTYPE html>
<html lang="en">
<head>
<link rel="stylesheet" href="mystyle.css">
<style>
h1{
color: orange;
}
</style>
</head>
<body>
<h1>This is heading</h1>
<p>This is paragraph</p>
</body>
</html>

Відображатиметься так:

Приклад. Однак, якщо внутрішній стиль визначено перед посиланням на зовнішню таблицю стилів, елементи <h1> будуть navy ("морськими"):

<!DOCTYPE html>
<html lang="en">
<head>
<style>
h1{
color: orange;
}
</style>
<link rel="stylesheet" href="mystyle.css">
</head>
<body>
<h1>This is heading</h1>
<p>This is paragraph</p>
</body>
</html>

Каскадний порядок

Який стиль використовуватиметься, якщо для елементи HTML задано більше ніж один стиль?
Усі стилі на сторінці "каскадуватимуть" у нову "віртуальну" таблицю стилів за такими правилами, де номер один має найвищий пріорітет:

  1. Вбудований стиль (всередині елемента HTML)
  2. Зовнішні та внутрішні таблиці стилів (у головному розділі)
  3. Браузер за замовчуванням

Таким чином, вбудований стиль має найвищий пріоритет і перекриває зовнішні та внутрішні стилі та стандартні налаштування браузера.
