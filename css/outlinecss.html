Контур CSS

Контур - це лінія, яку малюють навколо елементів поза межами, щоб зробити елемент "виділяючимся".

CSS має такі властивості структури:

  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

Примітка. Контур відрізняється від межі(рамки)! На відміну від рамки, контур малюється за межами елемента та може перекривати інший вміст. Крім того, контур НЕ є частиною розмірів елемента; ширина контуру не впливає на загальну ширину та висоту елемента.

Стиль контуру СSS

Властивість outline-style визначає стиль контуру та може мати одне з таких значень:

  • dotted - визначає пунктирний контур
  • dashed - визначає пунктирний контур
  • solid - визначає суцільний контур
  • double - визначає подвійний контур
  • groove - визначає 3D рифлений контур
  • ridge - визначає тривимірний ребристий контур
  • inset - визначає контур 3D-вставки
  • outset - визначає початковий контур 3D
  • none - не визначає контурів
  • hidden - визначає прихований контур

У наступному прикладі показано різні outline-style значення:

p.dotted {outline-style: dotted;}
p.dashed {outline-style: dashed;}
p.solid {outline-style: solid;}
p.double {outline-style: double;}
p.groove {outline-style: groove;}
p.ridge {outline-style: ridge;}
p.inset {outline-style: inset;}
p.outset {outline-style: outset;}
