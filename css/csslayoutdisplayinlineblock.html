Макет CSS - display:inline-block

display: inline-block значення

Порівняно з display:inline, основна видимість поляшає в тому, що display: inline-block дозволяє встановити ширину та висоту елемента.

Крім того, за допомогою display: inline-block верхні та нижні поля/відступи дотримуються, а за допомогою display: inline ні.

Порівняно з display:block, основна відмінність полягає в тому, що з display: inline-block після елемента не додається розрив рядка, тому елемент може розташовуватись поруч з іншими елементами.

У наступному прикладі показано різну поведінку display: inline, display: inline-block, diaplay: block :

span.a{
display: inline;
width: 100px;
height: 100px;
padding: 5px;
border: 1px solid blue;
background-color: yellow;
}
span.b{
display: inline-block;
width: 100px;
height: 100px;
padding: 5px;
border: 1px solid blue;
background-color: yellow;
}
span.c{
display: block;
width: 100px;
height: 100px;
padding: 5px;
border: 1px solid blue;
background-color: yellow;
}

Використання inline-blcok для створення навігаційних посилань

Одним із поширених способів використання display: inline-block є відображення елементів списку горизонатльно, а не вертикально. У наступному прикладі створюються горизонтальні навігаційні посилання:

.nav{
background-color: yellow;
list-style-type: none;
text-align: center;
padding: 0;
margin: 0;
}
.nav li{
display: inline-block;
font-size: 20px;
padding: 20px;
}
