Панелі навігації

Для будь-якого веб-сайту важлива проста у використанні навігація.

За допомогою CSS ви можете перетворити нудні HTML-меню на гарні навігаційні панелі

Панель навігації = список посилань

Для панелі навігації потрібен стандартной HTML як основа

У наших прикладах ми створимо панель навігації зі стандартного списку HTML.

Панель навігації - це в осонвному список посилань, тому використання елементів ul i li має сенс:

<ul>
<li><a href="#home">Home</a></li>
<li><a href="#news">News</a></li>
<li><a href="#contact">Contact</a></li>
<li><a href="#about">About</a></li>
</ul>

Відображатиметься так:

Тепер давайте видалимо маркери, поля та відступи зі списку:

ul{
list-style-type: none;
margin: 0;
padding: 0;
}

Відображатиметься так:

Пояснення.

  • list-style-type: none; - знімає кулі. Панелі навігації не потрібні маркери списку
  • встановіть margin:0; i padding:0; видаліть налаштування браузера за замовчуванням

Код у наведеному прикладі є стандартним кодом, який використовується як у вертикальній, так і в горизонтальній панелях навігації, про який ви дізнаєтесь більше в наступних розділах.
