Ширина контуру CSS

Властивість outline-width визначає ширину контуру та може мати одне з таких значень:

  • thin - 1 піксель
  • medium - 3 пікселі
  • thick - 5 пікселів
  • певний розмір - у px,pt,cm,em тощо.
p.ex1 {
border: 1px solid black;
outline-style: solid;
outline-color: red;
outline-width: thin;
}

p.ex2 {
border: 1px solid black;
outline-style: solid;
outline-color: red;
outline-width: medium;
}

p.ex3 {
border: 1px solid black;
outline-style: solid;
outline-color: red;
outline-width: thick;
}

p.ex4 {
border: 1px solid black;
outline-style: solid;
outline-color: red;
outline-width: 4px;
}
