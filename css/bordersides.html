Межа CSS - окремі сторони

З прикладів на попередніх сторінках ви побачили, що можна вказати різні межі для кожної сторони.

У CSS також є властивості для визначення кожної з меж (верхньої, правої, нижньої та лівої):

p{
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;
}

Якщо border-style властивість має чотири значення:

  • border-style: dotted solid double dashed;
    • вехня межа крапками
    • права межа суцільна
    • нижня межа подвійна
    • ліва межа пунктирна

Якщо border-style властивість має три значення:

  • border-style: dotted solid double;
    • вехня межа крапками
    • права та ліва межа суцільна
    • нижня межа подвійна

Якщо border-style властивість має два значення:

  • border-style: dotted solid;
    • вехня та нижня межа крапками
    • права та ліва межа суцільна

Якщо border-style властивість має одне значення:

  • border-style: dotted;
    • усі чотири межі крапками
body{
text-align: center;
}
p.four{
border-style: dotted solid double dashed;
}
p.three{
border-style: dotted solid double;
}
p.two{
border-style: dotted solid;
}
p.one{
border-style: dotted;
}
