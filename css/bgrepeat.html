Фонове повторення CSS

За замовчуванням background-image властивість повторює зображення як горизонтально, так і вертикально.
Деякі зображення слід повторювати тільки по горизонталі або вертикалі, інакше вони виглядатимуть дивно, ось так:

body{
background-image: url("img/bg1.jpg")
}

Якщо зображення вище повторюється лише горизонтально (background-repeat: repeat-x;), фон виглядатиме краще:

body {
background-image:url("bg1.jpg");
background-repeat: repeat-x;
}

Порада: Щоб повторити зображення вертикально, встановіть background-repeat: repeat-y;

СSS backgound-repeat: без повторення

Відображення фонового зображення лише один раз також визначається background-repeat властивістю: Приклад. Показувати фонове зображення лише один раз:

body {
background-image: url("img/bg2.jpg");
backgound-repeat: no-repeat;
}

У наведеному вище прикладі фонове зображення розміщене в тому самому місці, що й текст. Ми хочемо змінити положення зображення, щоб воно не надто заважало тексту.

СSS background-position

Властивість background-position використовується для визначення положення фонового зображення.

body{
background-image:url("img/bg2.jpg");
background-repeat: no-repeat;
background-position: right top;
}
