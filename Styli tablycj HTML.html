Styli tablycj HTML

Vykorystovujte CSS, schob vashi tablyci vygljadaly krasche.

Tablucja HTML - smugy zebry, vertykaljni smugy zevry,kombinovani

Jakscho vy dodaste kolir fonu do koznogo inshogo rjadka tablyci, cy otrymaete garnyj efekt smuzhok zebry.
Schob stylizuvaty kozen inshyj element rjadka tablyci, vykorystovujte :nth-child(even) selektor tak:
tr:nth-child(even) {
background-color: #ff0000;
}
Prymitka. jakscho vy vykorystovujete (odd) zamistj even, stulj byde vidbuvatysja v rjadky 1,3,5 toscho zamistj 2,4,6 toscho.

Schob zrobutu vertykaljni smygu zebry, stylizujte kozen dryguj stovpecj, a ne kozen drugyj rjadok.
td:nth-child(even),th:nth-child(even) {
background-color: #ff0000;
}
Prumitka. Rozmistitj :nth-child() selektor na oboh elementah th i td, jakscho vy hochete maty stylj dlja oboh zagolovkiv i zvychajnyh klitynok tablyci.

Vy mozete pojednatu styli z dvoh navedenyh vysche prykladiv, i vy matymete smugy v koznomy drugomy rjadku ta koznomu inshomu stovpci.
Jakscho vy vykorystovuete prozoryj kolir, vy otrymaete efekt nakladannja.
Vykorystovujte rgba schob vyznachutu prozoristj koljoru.
tr:nth-child(even) {
background-color: #ff0000;
}
td:nth-child(even),th:nth-child(even) {
background-color: #ff0000;
}

Goryzontaljni rozdiljnyky

Jakscho vy vkazete mezi lyshe vnyzu rjadka tablyci, u vas bude tablycja z goryzontaljnymy rozdiljnykamy.
Dodajte border-bottom vlastyvistj do vsih tr elementivmschob otrymaty goryzontaljni rozdiljuvachi:
tr {
border-bottom: 1px solid #0000ff;
}

Tablycja, scho obertajetjsja

Vykorystovujte :hover selektor, tr elementa schob vydilyty rjadky tablyci pry navedenni kyrsora myshi:
tr:hover {background-color:#0000ff}
