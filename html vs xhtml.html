HTML проти XHTML

HTML - це суворіща версія HTML на основі XML.

Що таке XHTML?

  • XHTML означає EXtensible Hyper Text Markup Language.
  • XHTML - це суворіша версія HTML на основі XML
  • XHTML - це HTML, визначений як програма XML
  • XHTML підтримується всіма основними браузерами

Чому XHTML?

XML - це мова розмітки, де всі документи мають бути правильно розмічені (бути добре сформованими).
XHTML був розвоблений, щоб зробити HTML більш розширюваним і гнучким для роботи з іншими форматами даних (наприклад XML). Крім того, браузери ігнорують помилки на сторінках HTML і намагаються відобразити веб-сайт, навіть якщо на ньому є помилки в розмітці. Тому XHTML має набагато суворішу обробку помилок.

Найважливіші відмінності від HTML

  • !DOCTYPE є обов'язковим
  • Атрибут xmlns в html є обов'язковим
  • html, head, title, body є обов'язковими
  • Елементи завжди повинні бути правильно вкладеними
  • Елементи завжди повинні бути закриті
  • Елементи завжди повинні бути в нижньому регістрі
  • Назви атрибутів зажди повинні бути в нижньому регістрі
  • Значення атрибутів завжди повинні бути в лапках
  • Мінімізація атрибутів заборонена
