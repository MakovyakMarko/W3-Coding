Scho take CSS?

CSS vykorystovuetjsja dlja formatuvannja maketa web-storinky.
Za dopomogoju CSS vy mozete keruvaty koljorom, shryftom,rozmirom tekstu,intervalom miz elementamu, roztachuvannjam elementiv, jaki fonovi zobrazennja abo koljory fonu vykorystovuvatymutjsja, riznymy dysplejamy dlja riznyh prystrojiv i rozmiriv ekrana, a takoz nabagato biljshe!
Porada: clovo kaskadnuj oznachae, scho stylj, zastosovanuj do batjkivsjkogo elementa, takoz zastosovuvatymetjsja do vsih dochirnyh elementiv u batjkivsjkomu elementi. Otze, jakscho vy vstanovyte kolir ocnovnogo tekstu na sunij, usi zagolovky, abzacy, ta inshi tekstovi elementy v osnovnomu teksti takoz otrymajutj toj samyj kolir (jakscho vu ne vkazete schosj inshe)!

Vykorystannja CSS

CSS mozna dodaty do documentiv HTML trjoma sposobamy:

  • Inline - za dopomogoju style atrybuta vseredyni elementiv HTML
  • Vnytrishnij - za dopomogoju style elementa v head rozdili
  • Zovnishnij - za dopomogoju link elementa dlja posylannja na zovnishnij fail css

Najposhyrenishym sposobom dodavannja CSS je zberezennja styliv u zovnishnih failah CSS. Odnak u cjomy pidruchnyku my budemo vykorystovuvaty vbydovani ta vnutrishni styli, tomu scho ce legshe prodemonstruvaty, a vam legshe sprobuvaty samostijno,

Zovnishnij CSS

Zovnishnja tablycja styliv vykorystovuetjsja dlja vuznachennja stylju bagatjoh storinok html.
Schob vykorystovyvaty zovnishnju tablycju styliv, dodajte posulannja na neji v head rozdili storinky html.

Porada: Za dopomogoju zovnishnjoji tablyci styliv vu mozete zminyty zovnishnij vugljad vsjogo web-saitu, zminuvshu odun file!

  • Vykorystovujte color vlastyvistj CSS dlja koljoru tekstu
  • Vykorystovujte font-family vlastyvistj CSS dlja tekstovyh shryftiv
  • Vykorystovujte font-size vlastyvistj CSS dlja rozmiriv tekstu
  • Vykorystovujte border vlastyvistj CSS dlja mez
  • Vykorystovujte padding vlastyvistj CSS dlja prostoru vseredyni mezi
