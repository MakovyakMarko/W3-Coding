Neuporjadkovani spysky HTML

Teg HTML ul vyznachae nevporjadkovanyj (markovanyj) spysok.

Neuporjadkovanuj spysok HTML

Neyporjadkovanyj spysok pochynaetjsja z ul tega. Kozen element spysku pochynaetjsja z li tegu.
Elementy spysky bydytj poznacheni markeramy za zamovchuvannjam:

  • Coffee
  • Tea
  • Milk

Neuporjadkovanyj spysok HTML - vyberitj marker elementa spysku

Goryzontaljnyj spysok iz CSS

Spysky HTML mozna stylizuvaty riznymy sposobamy za dopomogoju CSS.
Odnym iz populjarnyh sposobiv je stvorenja stylju spysky po goryzontali, scho stvoryty navigacijne menju:

style
ul{
list-style-type:none;
margin:0;
padding:0;
overflow:hidden;
background-color:#333333;
}
li{
float:left;
}
li a{
display: block;
color: white;
text-align: center;
padding:16px;
text-decoration:none;
}
li a:hover{
background color:red;
}
/style

Pidsumky rozdilu

  • Vykorystovujte ul element HTML, schob vyznachyty nevporjadkovanyj spysok
  • Vykorystovujte list-style-type vlastyvistj CSS, schob vyznachyty marker elementa spysku
  • Spysky mozytj buty vkladenymy
  • Elementy spysku mozytj mistyty inshi elementy html
  • Vykorystovujte vlastyvistj CSS float:left schob vidobazyty spysok po goryzontali
