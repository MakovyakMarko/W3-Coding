Графіка HTML SVG

SVG визначає векторну графіку у форматі XML.

Що таке SVG

  • SVG означає масштабовану векторну графіку
  • SVG використовується для визначення графіки для Інтернету
  • SVG є рекомендацією W3C

Елемент HTML <svg>

Елемент HTML <svg> є контейнером для графіки SVG.
SVG має кілька методів малювання шляхів, прямокутників, кіл, тексту та графічних зображень

Коло SVG

<svg width="100" height="100" > <circle
cx="50"
cy="50"

stroke="blue"
stroke-width="4"

</svg>

Відмінності між SVG і Canvas

SVG - це мова для опису 2d-графіки в XML.
Canvas малює 2D-графіку на льоту (за допомогою JavaScript).
SVG базується на XML, що означає, що кожен елемент доступний у SVG DOM. Ви можете приєднати обробники подій JavaScript до елемента.
У SVG кожна намальована фігура запам'ятовується як об'єкт. Якщо атрибути об'єкта SVG змінено, бравзер може атоматично потвторно відобразити форму.
Полотно відтворюється піксель за пікселем. У Canvas, як тільки графіка намальована, вона забувається бравзером. Якщо його положення потрібно змінити, необхідно перемалювати всю сцену, включаючи будь-які об'єкти, які могли бути покриті графікою.
