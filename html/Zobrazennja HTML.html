Zobrazennja HTML

Zobrazennja mozutj pokrschyty dyzain i zovnishnij vygljad web-storinky.



Syntaksys zobrazenj HTML

Teg HTML img vykorystovuetjsja dlja vbydyvannja zobrazennja u web-storinku.
Zobrazennja tehnichno ne vstavljajutjsja na web-storinku, zobrazennja posylajutjsja na web-storinky.
Teg img stvorjue prostir dlja utrymannja zobrazennja,na jake posylajetjsja.
Teg img poroznij, vin mistytj lyshe atrybuty i ne mae zalryvaljnogo tegu.
Teg img mae dva obovjazkovi atrybuty:

  • src - Vkazue shljah do zobrazennja
  • alt - Vuznachae alternaryvnyj tekst dlja zobrazennja

Syntaksys: img src="URL" alt="alternatetext"

Atrybut src

Neobhidnyj src atrybut vyznachae shljah (URL) do zobrazennja.
Prumitka. Koly web-storinka zavantazuetjsja, same brauzer v cej moment otrymue zobrazennja z web-servera ta vstavljae jogo na storinku. Tomu perekonajtesja, scho zobrazennja faktychno zalyshaetjsja na tomu samomu misci po vidnoshennja do web-atorinky, inakshe vachi vidviduvachi otrymajutj znachok nepracezdatnosti posylannja. Znachok nepracjujuchogo posylannja ta alt tekst vidobrazajutjsja, jakscho brauzer ne moze znajty zobrazennja.

Atrybut alt

Neobhidnyj alt atrybut nadaje alternatyvnyj tekst dlja zobrazennja, jakscho korystuvach z jakyhosj prychyn ne moze jogo peregljanuty (cherez poviljne zjednannja, pomylky v atrybuti src abo jakscho korystuvach vykorystovuje programy zchytuvannja z ekrana).

Pruklad

Porada. Programa zchytuvannja z ekrana - ce programa, jaka zchytuje HTML kod i dozvoljae korystuvachevi "prosluhaty" vmist. Programy zchytuvannja z ekrana korysni dlja ljudej z vadamy zoru abo navchannjam.

Rozmir zobrazennja - shyryna i vysota

Vu mozete vykorystovuvaty style atrybut dlja vyznachennja shyryny ta vysoty zobrazennja.
img src="1.jpg" alt="Ukraine" style="width:500px;height:600px;"
Jak variant vy mozete vykorystovuvaty width i height atrybuty:
img src="1.jpg" alt="Ukraine" width="500" height="600"
Atrybuty width i height zavzdy vyznachajutj shyrynu i vysotu zobrazennja v pikseljah.
Prymitka. Zavzdy vkazujte shyrynu ta vusotu zobrazennja. Jakscho shyryna ta vysota ne vkazani, web-storinka moze blymaty pid chas zavantazennja zobrazennja.

Shyryna i vysota chu stylj?

Usi atrybuty width i height i style dijsni v HTML.
Prote mu proponujemo vykorystovyvaty style atrybut. Ce zapobigae zmini rozmiry zobrazennja tablycjamy styliv.

Zobrazennja v inshij papci

Jakscho y vas e zobrazennja v pidpapci, vu povynni vkljuchyty nazvu papky v src atrybyt: img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;"

Zobrazennja na inshomu web-serveri/web-sajti

Dejaki web-saity vkazujutj na zobrazennja na inshomy serveri.
Schob vkazuvaty na zobrazennja na inshomu serveri, neobhidno vkazaty absoljutnu (povnu) URL-adresu s src atrybuti:
img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools"

Animovani zobrazennja

HTML dozvoljae stvorjuvaty animovani GIF-faily:
img src="guse.gif" alt="Guse" width="270" height="400"

Pisdumok rozdilu

  • Vykorystovujte element HTML img,schob vyznachyty zobrazennja
  • Vykorystovujte src atrybut HTML, schob vyznachyty URL-adresu zobrazennja
  • Vykorystovujte alt atrybut HTML, schob vyznachyty alternaryvnyj telst dlja zobrazennja, jakscho jogo nemozlyvo vidobrazyty
  • Vykorystovujte HTML width i height atrybut abo CSS width i height vlastyvosti, schob vuznachyty rozmir zobrazennja
  • Vykorystovujte vlastyvistj CSS float, schob zobrazennja plavalo pravoruch abo livorych
