HTML JavaScript

JavaScript robytj storinky HTML biljsh dynamichnymy ta interaktyvnymy.

Teg HTML script

Teg html script vykorystovuetjsja dlja vyznachennja klientsjkogo scenariju (scrypta).
Element script abo mistytj operatory scenariju, abo vkazyje na zovnishnij fajl scenariju cherez src atrybut.
Poshyrenym vykorystannjam JavaScript je manipyljuvannja zobrazennjamy, perevirka formy ta dynamichni zminy vmistu.
Schob vybraty element HTML, JavaScript najchastishe vykorystovyje document.getElementById() metod.
