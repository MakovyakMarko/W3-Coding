Відео HTML

Елемент HTML <video> використовується для показу відео на веб-сторінці.

<video width="400" controls>
<source src="mov_bbb.mp4" type="video/mp4" />
</video>

Як це працює

Атрибут controls додає елементи керування відео, як-от відтворення, пауза та гучність.
Рекомендується завжди включати атрибути width i height. Якщо висоту та ширину не встановлено, сторінка може мерехтіти під час завантаження відео.
Елемент <source> дозволяє вказати альтенативні відеофайли, які браузер може вибрати. Браузер використовуватиме перший розпізнаний формат.
Текст між тегами <video> і </video> відображатиметься лише в браузерах, які не підтримують цей video елемент.

Автоматичне відтворення video HTML

Для атоматичного запуску відео використовувайте autoplay атрибут

<video width="320" height="240" autoplay>
<source src="movie.mp4" type="video/mp4">
Ваш браузер не підтримує <video> tag. </video>

Додайте muted після autoplay, щоб відео почало відтворюватись автоматично (але без звуку).

<video width="320" height="240" autoplay muted>
<source src="movie.mp4" type="video/mp4">
Ваш браузер не підтримує <video> tag. </video>

Відео HTML - методи, властивості і події

HTML DOM визначає методи, властивості та події для video елемента.
Це дозволяє завантажувати, відтворювати та призупиняти відео, а також встановлювати тривалість і гучність.
Існують також події DOM, які можуть сповіщати вас про початок відтворення відео, його призупинення тощо.

<div style="text-align:center">
<button onclick="playPause()">Play/Pause</button>
<button onclick="makeBig()">Big</button>
<button onclick="makeSmall()">Small</button>
<button onclick="makeNormal()">Normal</button>

<video id="video1" width="420">
<source src="mov_bbb.mp4" type="video/mp4">
</video>
</div>
<script>
var myVideo= document.getElementById("video1");
function playPause(){
if (myVideo.paused)
myVideo.play();
else
myVideo.pause();
}
function makeBig(){
myVideo.width=560;
}
function makeSmall(){
myVideo.width=320;
}
function makeNormal(){
myVideo.width=420;
}
</script>
