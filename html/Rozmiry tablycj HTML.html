Rozmiry tablycj HTML

Tablyci HTML mozytj maty rizni rozmiry dlja koznogo stvpcja, rjadka abo vsieji tablyci.
Vykorystovujte style atrybut iz vlastyvostjamy width i height dlja vyznachennja rozmiru tablyci, rjadka chy stovpcja.
