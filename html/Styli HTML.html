Стилі HTML

Атрибут HTML style використовується для додавання стилів до елемента таких як колір, шрифт, розмір тощо. Приклад

<p>Я звичайний</p>
<p style="color:red;">Я червоний</p>
<p style="color:blue;">Я синій</p>
<p style="font-size:50px;">Я ВЕЛИКИЙ</p>

Відображатиметься так:

Я звичайний

Я червоний

Я синій

Я ВЕЛИКИЙ

Атрибут стилів HTML

Атрибут HTML style має такий синтаксис:
<tagname style="property:value" >

Властивість є властивістю CSS. Значення - це значення CSS.

<h2 style="background-color:powderblue;">Колір фону</h2>
<p>CSS background-color властивість, що визначає колір фону для елемента HTML</p>
<p style="background-color:tomato;"> Якщо добавити в елемент <body> style="background-color: rgb(5, 140, 252);" зробить колір фону голубим</p>

Відображатиметься так:

Колір фону

CSS background-color властивість, що визначає колір фону для елемента HTML

Якщо добавити в елемент <body> style="background-color: rgb(5, 140, 252);" zrobuv kolir fony golubym

Колір тексту

Властивість CSS color визначає колір тексту для елементу HTML:

<h1 style="color:red;">Це червоний заголовок</h1>
<p style="color:rgb(238, 196, 27);"> Це золотий параграф</p>

Відображатиметься так:

Це червоний заголовок

Це золотий параграф

Шрифти

Властивість CSS font-family визначає шрифт, який буде використовуватися для елеметна HTML:

<h1 style="font-family:Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif;">Це заголовок</h1>
<p style="font-family:'Arial';">Це параграф</p>

Відображатиметься так:

Це загловок

Це параграф

Розмір тексту

Властивість font-size визначає розмір тексту для елемента HTML

<h1 style="font-size: 300%;">Це заголовок</h1> <p style="font-size: 160%;">Це параграф</p>

Відображатиметься так:

Це заголовок

Це параграф

Вирівнювання тексту

Властивість CSS text-align визначає горизонтальне вирівнювання тексту для елемента HTML:

<h1 style="text-align: center;">Це заголовок</h1> <p style="text-align: right;">Це параграф</p>

Відображатиметься так:

Це заголовок

Це параграф

  • Використовуйте style атрибут для стилізації елементів HTML
  • Використовуйте background-color для кольору фону
  • Використовуйте color для кольорів тексту
  • Використовуйте font-family для текстових шрифтів
  • Використовуйте font-size для розмірів тексту
  • Використовуйте text-align для вирівнювання тексту
