Mezi tablyci html

Tablyci html mozutj maty mezi riznyh styliv i form

Jak dodaty mezu

Koly vy dodaete kordon do tablyci, vy takoz doaete mezi navkolo koznoki klitynky tablyci:

A B C
Aa Bb Cc
Dd Ee Ff

Schob dodaty mezu, vykorystovujte border vlastyvistj CSS dlja elementiv table,th,td

Zgornuti mezi tablyci

Schob unyknuty podvijnyh mez, jak u pryladi vysche, ystanovitj dlja border-collaps vlastyvosti CSS znachennja collapse

Mezi tablycj styliv

Aa Bb Cc
Dd Ee Ff
