HTML атрибути

Атрибути надають додаткову інформацію про елементи HTML.

  • Усі елементи HTML можуть мати атрибути.
  • Атрибути надають додаткову інформацію про елементи.
  • Атрибути завжди вказуються в початковому тегу.
  • Атрибут зазвичає представлені парами ім'я/значення, наприклад: "name="value"".

Атрибут href

Тег <a> визначає гіперпосилання. Атрибут href визначає URL-адресу сторінки, на яку переходить посилання. Наприклад:

<a href="https://www.google.com">Visit Google</a>

Відображатиметься так:

Visit Google

Атрибут src

Тег <img> використовується для вбудування зображення на сторінку HTML. Атрибут src визначає шлях до зображення, яке має відображатися:

<img src="3.jpeg" width="400" height="270">

Є два способи вказати URL-адресу в src атрибуті:

  • Абсолютна URL-адреса. Посилання на зовнішнє зображення, розміщене на іншому веб-сайті. Приклад: <src="https://media-exp1.licdn.com/dms/image/company-logo.jpg"> Примітки. Зовнішні зображення можуть бути захщені авторським правом. Якщо ви не отримаєте дозволу на його використання, можливо, ви порушуєте закони про авторські права. Крім того, ви не можете контролювати зовнішні зображення, які можуть раптово видалити або змінити.
  • Відносна URL-адреса - посилання на зображення, розміщене на вашому веб сайті, в корневому каталозі або підпапці. Тут URL-адреса не містить доменного імені. Якщо URL-адреса починається без косої риски, вона буде відносно поточної сторінки. Приклад: src="1.jpg". Якщо URL-адреса починається з косої риски, вона буде відносно домену. Приклад: src="/images/img_girl.jpg" Порада. Майже завжди краще використовувати відносні URL-адреси. Вони не зламаються, якшо ви змінете домен.

Атрибути ширини та висоти

Атрибут alt

Обов'язковий alt атрибут для тегу <img> визначає альтернативний тексту для зображення, якщо зображення з якихось причин не може бути відображено. Це може бути пов'язано з повільним з'єднанням, помилкою в src атрибуті або використанням програми зчитування з екрана. Приклад:

<img src="10.jpg" alt="Ukraine" >

Атрибут style

Атрибут style використовується для додавання стилів до елемента, таких як колір, шрифт, розмір тощо.Приклад:

<p style="color: red;">This is a red paragraph.</p>

Відображатиметься так:

This is a red paragraph.

Атрибут lang

Ви завжди повинні ключати lang атрибут всередині html тегу, щоб огологувати мову веб-сторінки. Це покликане допомогти пошуковим системам і браузерам. Коди країн також можна додатти до коду мови в lang атрибуті. Отже, перші дви символи визначають мову сторінки HTML, а останні два Символи визначають країну. Ці теги вказані на початку сторінки.

<!DOCTYPE html>
<html lang="ua-UA">
<body>
...
</body>
</html>
<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

Атрибут заголовка

<p title="Що таке атрибут title?">

Відображатиметься так:

Атрибут title визначає додаткову інформацію про елемент. Значення атрибута title буде відображатися як підказка, коли наведете на елемент. Наприклад наведіть курсор на цей абзац.

Ми радимо завжди використовувати атрибути нижнього регістру.

Ми пропонуємо: завжди вказувати значеня атрибутів

Стандарт HTML не вимагає лапок навколо атрибутів. Однак ми рекомендуєм лапки в HTML, вони вимагаєються для суворіших типів дукоментів, таких як XHTML.
<href="https://www.w3schools.com/html/"> - добре.
<href=https://www.w3schools.com/html/> - погано.

Іноді доводиться використовуват лапки. У цьому прикладі атрибут title відображається неправильно, оскільки він містить пробіл:
<p title=About W3Schools>
Правильно так - <p title="About W3Schools">

Одинарні чи подвійні лапки?

Подвійні лапки навколо значень атрибутів є найпоширенішими в HTML, але також можна використовувати одинарні лапки. У деяких ситуаціях, коли саме значення атрибута містить подвійні лапки, необхідно використовувати одинарні лапки:
<p title='John "ShotGun" Nelson'>,
або навпаки <p title="John 'ShotGun' Nelson">

Підсумок розділу

  • Усі елементи HTML можуть мати атрибути.
  • Атрибут href визначає URL-адресу сторінки, на яку переходить посилання.
  • Атрибут src в <img> вказує шлях до зображення, яке має відобразитися.
  • Атрибут height i width в <img> надають інформацію про розмір зображення (висоту і ширину).
  • Атрибут alt надає <img> альтернативний текст для зображення.
  • Атрибут style використовується для додавання стилів до елемента, таких як колір, шрифт, розиір тощо.
  • Атрибут lang тегу <html> оголошує мову веб-сторінки.
  • Атрибут title визначає додаткову інформацію про елемент.
