HTML плагіни

Плагіни - це комп'ютерні програми, які розширюють стандартні функції браузера.

Плагіни

Плагіни були розроблені для використання в різних цілях:

  • для запуску Java-аплетів
  • Для запуску елементів керування Microsofr ActioveX
  • Для відображення флещ-фільмів
  • Для відображення карт
  • Для сканування на віруси
  • Для підтвердження банківського ідентифікатора

УВАГА!
Більшість браузерів більше не підтримують аплети ти плагіни Java.
Елементи керування ActiveX більше не підтримуються жодним браузером.
У сучасних браузерах також відключено підтримку Shockwave Flash.

Елемент <object>

Елемент <object> підтримується всіма браузерами.
Елемент <object> визначає вбудований об'єкт у документ HTML.
Його розроблено для вбудування плагінів (наприклад, Java-аплетів, програм для читання PDF-файлів і програвачів Flash) у веб сторінки, але також можна використовувати для включення HTML у HTML.

<object width="100%" height="500px" data="snippet.html"></object>

Або зображення:

<object data="audi.jpeg"> </object>

Елемент <embed>

Елемент <embed> підтримується в усіх основних браузерах.
Елемент <embed> також визначає вбудований об'єкт у документ HTML.
Веб-браузери вже давно підтримують елемент <embed>. Однак він не був частиною специфікації HTML до HTML5. Елемент <embed> також можна використовувати для включення HTML у HTML.

<embed src="audi.jpeg">

Зауважте, що елемент <embed> не має закриваючого тегу. Він не може містити альтернативний текст.
