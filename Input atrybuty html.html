Input атрибути HTML

У цій главі описуються різні атрибути для <input> елемента HTML.

Атрибут value

Атрибут input value визначає початкове значення для поля введення:

<form>
<label for="fname">Ім`я:</label>
<input type="text" id="fname" name="fname" value="John">
<label for="lname">Фамілія:</label>
<input type="text" id="lname" name="lname" value="Doe">

Атрибут readonly

Атрибут input readonly вказує, що поле введення доступне лише для читання.
Поле введення лише для читання не можна змінити(однак користувач може перейти до нього табуляцією, виділити його та скопіювати з нього текст).
Значення поля введення лише для читання буде надіслано під час надсилання форми!

<form>
<label for="fname">Ім`я:</label>
<input type="text" id="fname" name="fname" value="John" readonly>
<label fot="lname">Фамілія:</label>
<input type="text" id="lname" name="lname" value="Doe"> </form>

Атрибут disabled

Атрибут input disabled вказує, що поле введення має бути вимкнено.
Вимкнене поле введення непридатне для використання та на нього не можна натискати.
Значення вимкненого поля введення не буде надіслано під час надсилання форми!

<form>
<label for="fname">Ім`я:</label>
<input type="text" id="fname" name="fname" value="John" disabled>
<label for="lname">Фамілія:</label>
<input type="text" id="lname" name="lname" value="Doe">

Атрибут розміру:

Атрибут input size визначає видиму ширину поля введення в символах.
Значення за замовчуванням size-20.
Примітка. size атрибут працює з такими типами введення: текст, пошук, телефон, url, електронна пошта, пароль.

<form> <label for="fname">Ім`я:</label><br/>
<input type="text" id="fname" size="50"><br/>
<label for="pin">PIN:</label><br/>
<input type="text" id="pin" name="pin" size="4">
</form>

Атрибут maxlength

Атрибут input maxlength визначає максимальну кількість символів встановлену в полі введення.
Примітка. Якщо maxlength встановлено а, поле введення не прийматиме більше вказаної кількості символів. Однак це атрибут не забезпечує зворотнього зв`язку. Отже, якщо ви хочете попередити користувача, ви повиння написати код JavaScript.

<form> <label for="fname">Ім`я:</label><br/>
<input type="text" id="fname" size="50"><br/>
<label for="pin">PIN:</label><br/>
<input type="text" id="pin" name="pin" maxlength="4" size="4">
</form>

Атрибути min i max

Вхідні дані min та max атрибути вказують мінімальне та максимальне значення для поля введення.
Атрибути min та max працюють із такими типами введення: число, діапазон, дата, дата-час-локальний, місяць, час і тиждень.
Порада. Використовуйте атрибути min і max разом, щоб створити діапазон допустимих значень.

<form>
<label for="datemin">Введіть дату після 1990-01-01:</label>
<input type="date" id="datemin" name="datemin" min="1990-01-01"> <br/> <br/>

<label for="datemax">Введіть дату до 2022-07-14:</label>
<input type="date" id="datemax" name="datemax" max="2022-07-14"> <br/> <br/>

<label for="quantity">Кількість (між 1 та 5):</label>
<input type="number" id="quantity" name="quantity" min="1" max="5">
<input type="submit" value="Надіслати"> </form>

Атрибут multiple

Атрибут input multyple вказує, що користувачеві дозволено вводити більше одного значення в поле введення. Атрибут multyple працює з такими типами введення: електронна пошта та файл.

<form> <label for="files">Виберіть файли:</label> <input type="file" id="files" name="files" multiple> </form>

Атрибут pattern:

Атрибут input pattern визначає регулярний вираз, за яким перевіряється значення поля введення під час надсилання форми.
Атрибут pattern працює з такими типами введення: текст, дата, пошук, URL-адреса, телефон, електронна пошта, пароль. Порада. Використовуйте глобальний атрибут title, щоб описати шаблон, щоб допомогти корстувачеві.

<form>
<label for="country_code">Код країни:</label>
<input type="text" id="country_code" name="country_code" pattern="[A-Za-z]{3}" title="Три букви коду країни">
</form>

Атрибут placeholder

Атрибут inout placeholder виначає коротку підказку, яка описує очікуване значення поля введення (приклад значення або короткий опис очікуваного формату).
Коротка підказка відображається в полі введення перед тим, як користувач введе значення.
Атрибут placeholder працює з такими типами введення: текст, пошук, URL-адреса, телефон, електронна пошта та пароль.

<form>
<label for="phone">Введіть номер телефону:</label> <input type="tel" id="phone" name="phone" placeholder="+38-xxx-xxx-xx-xx" pattern="+38-[0-9]{3}-[0-9]{3}-[0-9]{2}-[0-9]{2}"> </form>

Атрибут required

Атрибут input required визначає, що поле введення має бути заповнене перед надсиланням форми.
Атрибут required працює з такими типами введення: текст, пошук, URL-адреса, телефон, електронна пошта, пароль, засоби вибору дати, номер, прапорець, радіо та файл.

<form>
<label for="username">Username:</label>
<input type="text" id="username" name="username" required>
</form>

Атрибут step

Атрибут input step визначає допустимі інтервали чисел поля введення.
Приклад: якщо step="3", допустимі числа -3, 0, 3, 6 тощо.
Порада: цей атрибут можна використовувати разом із атрибутами min i max для створення діапазону допустимих значень.
Атрибут step працює з такими типами введення: число, діапазон, дата, дата-час-локальний, місяць, час і тиждень.

<form>
<label for="points">Очки:</label>
<input type="number" id="points" name="points" step="3">
</form>

Атрибут autofocus

Атрибут input autofocus визначає, що поле введення має автоматично отримувати фокус під час завантаження сторінки.

<form>
<label for="fname">Ім`я:</label>
<input type="text" id="fname" name="fname" autofocus>
<label for="lname">Фамілія:</label>
<input type="text" id="lname" name="lname">
</form>

Атрибути height i width

Вхідні дані height i width атрибути визначають висоту та ширину <input type="image"> елемента.
Порада. Завжди вказуйте атрибути висоти та ширини для зображень. Якщо встановлено висоту та ширину, місце, необхідне для зображення, резервується під час завантаження сторінки. Без цих атрибутів браузер не знає розміру зображення та не може зарезервувати для нього відповідний простір. В результаті макет сторінки буде змінюватись під час завантаження (під час завантаження зображення).

<form>
<label for="fname">Ім`я:</label>
<input type="text" id="fname" name="fname">
<label for="lname">Фамілія:</label>
<input type="text" id="lname" name="lname">
<input type="image" src="knopka.gif" alt="Submit" width="48" height="48">
</form>

Атирбут autocomplete

Атрибут input autocomlete визначає, увімкнути чи вимкнути автозаповнення для форми чи поля введення.
Автозаповнення дозволяє браузеру передбачити значення. Коли користувач починає вводити текст у поле, браузер має відобразити варіанти заповнення поля на основі раніше введених значень.
Атрибут autocomplete працює з <formm> такими input типами: текст, пошук, url, tel, електронна пошта, пароль, засоби вибору дати, діапазон і колір.

<form action="/action_page.php" autocomplete="on">
<label for="fname">First name^</label>
<input type="text" id="fname" name="fname">

<label for="lname">Last name</label>
<input type="text" id="lname" name="lname">

<label for="email">Email</label>
<input type="email" id="email" name="email" autocomplete="off">

<input type="submit" value="Submit">
</form>
