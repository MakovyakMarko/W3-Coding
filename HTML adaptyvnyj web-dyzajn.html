HTML adaptyvnuj web-dyzajn

Adaptyvnyj web-dyzajn - ce stvorennja web-storinok, jaki dobre vygljadajutj na vsih prystrojah!
Adaptyvnyj web-dyzajn avtomatychno pidlashtovujetjsja pid rizni rozmiry ekrana ta vikna peregljadu.

Scho take adaptyvnyj web-dyzajn?

Adaptyvnyj web-dyzajn ce vykorystannja HTML i CSS dlja avtomatychnogo zminy rozmiru, pryhovuvannja, skorotchennja abo zbiljshennja web-sajtu, schob vin dobre vygljadav na vsih prystrojah(nastiljnyh kompjuterah, planchetah i telefonah):

Nalashtuvannja vikna peregljadu

Schob stvoryty adaptyvnyj web-sajt, dodajte takyj meta teg do vsih svojih web-storinok: meta rel="viewport" content="width=device-width, initial-scale=1.0"
Ce vstanovytj oblastj peregljadu vashoji storinky, jaka dastj brauzery instrukciji schodo keruvannja rozmiramy ta mashtabuvannjam storinky.

Vidpovidni zobrazennja

Adaptyvni zobrazennja -ce zobrazennja, jaki dobre mashtabujutjsja, schob vidpovidaty bydj-jakomu rozmiru brauzera.

Vykorystannja vlastyvosti shyryny

Pokazuvaty rizni zobrazennja zalezno vid shyryny brauzera

Element html picture dozvoljaje vyznachyty rizni zobrazennja dlja riznyh rozmiriv vikon brauzera. Zminitj rozmir vikna brauzera, schob pobachyty, jak zminjuetjsja zobrazennja nuzche zalezno vid shyryny:

Adaptyvnyj rozmir tekstu

Rozmir tekstu mozna vstanovyty za dopomogoju odynyci "vw", scho oznachaje (viewport width) "shyrynu oblasti peregljadu".
Takym chynom, rozmir tekstu vidpovidatyme rozmiru vikna brauzera:

Viewport- ce rozmir vikna brauzera. 1vw=1% shyryny vikna peregljadu. Jakscho syryna peregljadu stanovutj 50sm, 1vw dorivnuje 0,5 sm.

Media zapyty

Okrim zminy rozmiru takstu ta zobrazenj, na adaptyvnyh web-storinkah takoz zvychajnym je vykorystannja media-zapytiv.
Za dopomogoju media-zapytiv vy mozete vyznachyty absoljutno rizni styli dlja riznyh rozmiriv brauzera.
Pruklad: zminitj rozmir vikna brauzera, wchob pobachyty, scho try div elementy, navedeni nuzche, vidobrazatymutjsja goryzontaljno na velykyh ekranah i vertykaljno na malenjkyh ekranah:
