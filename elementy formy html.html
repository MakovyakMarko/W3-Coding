Елементи форми HTML

У цьому розділі описано всі елементи форми HTML.

Елементи HTML <form>

Елементи HTML <form> може містити один або кілька таких елементів форми:

  • <input>
  • <label>
  • <select>
  • <textarea>
  • <button>
  • <fieldset>
  • <legend>
  • <datalist>
  • <output>
  • <option>
  • <optgroup>

Елемент <input>

Одним з найбільш часто використовуваних елементів форми є <input> елемент.
Елемент <input> можна відобразити кількома способами, залежно від type атрибута.

<form atcion="/action_page.php">
<label for="fname">Ім'я:</label>
<input type="text" id="fname" name="fname">
<input type="submit" value="Відправити">
</form>

Відображається так:

Ім'я:


Усі значення атрибута type розглянуто в наступному розділі.

Елемент <label>

Елемент <label> визначає мітку для кількох елементів форми.
Елемент <label> корисний для користувачів програми зчитування з екрана, оскільки програма зчитуваня з екрана вголос прочитає мітку, коли користувачів зосередиться на елементі введення.
Елемент <label> також допомагає користувачам, яким важко клацнути на дуже маленьких ділянках (наприклад перемикачі чи прапорці), оскільки, коли користувач натискає текст всередині <label> елемента, він перемикає перемикач/прапорець.
Атрибут for тега має бути рівним id атрибуту input елемента, щоб зв'язати їх разом

Елемент <select>

Елемент <select> визначає розкривний список:

<form action="/action_page.php">
<label for="cars">Виберіть мащину</label>
<select id="cars" name="cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>
<input type="submit" value="Надіслати"> </form>

Відображається так:

Елемент <option> визначає параметр, який можна вибрати.
За замовчуванням вибирається перший пункт у випадаючому списку.
Щоб визначити попередньо вибрану опцію, додайте selected атрибут до параметра:

<option value="audi" selected>Audi</option>

Видимі значення

Використовуйте size атрибут, щоб указати кількість видимих значень:

<form action="/action_page.php">
<label for="cars">Виберіть мащину</label>
<select id="cars" name="cars" size="3">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>
<input type="submit" value="Надіслати"> </form>

Дозволити кілька варіантів:

Використовуйте multiple атрибут, щоб дозволити користувачеві вибрати більше одного значення:

<form action="/action_page.php">
<label for="cars">Виберіть мащину</label>
<select id="cars" name="cars" size="4" multiple>
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="fiat">Fiat</option>
<option value="audi">Audi</option>
</select>
<input type="submit" value="Надіслати">
</form>

Елемент <textarea>

Елемент <textarea> визначає багаторядкове поле введення (текстову область):

<form action="action_page.php">
<textarea name="messsage" rows="10" cols="30">Кіт грався в саду</textarea>
<input type="submit">
</form>

Відображається так:



Атрибут rows визначає видиму кількість рядків у текстовій області.
Атрибут cols визначає видиму ширину текстової області.
Ви можете визначити розмір текстової області за допомогою CSS:

<textarea name="message" style="width:100%; height:200px;> Кіт грався в саду </textarea>

Елемент <button>

Елемент <button> визначає кнопку, яку можна натиснути:

<button type="button" onclick="alert('Hello World!')> Click me! </button>

Відображається так:

Примітка. Завжди вказуйте type атрибут для елемента кнопки. Різні браузери можуть використовувати різні типи за замовчуванням для елемента кнопки.

Елементи <fieldset> i <legend>

Елемент <filedset> використовується для групування пов'язаних даних у формі.
Елемент <legend> визначає підпис для <fieldset> елемента.

<form action="action_page.php">
<fieldset>
<legend>Песоналії</legend>
<label for="fname">Ім'я:</label;> <br/>
<input type="text" id="fname" name="fname" value="John"> <br/>
<label for="lname">Фамілія:</label> <br/>
<input type="text" id="lname" name="lname" value="Doe"> <br/> <br/>
<input type="submit" value="Submit">
</fieldset>
</form>

Елемент <datalist>

Елемент <datalist> визначає список попередньо визначених параметрів для <input> елемента.
Під час введення даних користувачі бачитимуть спадний спиок із попередньо визначеними параметрами.
Атрибут list елемента input повинен посилатися на id атрибут datalist елемента.

<form action="/action_page.php">
<input list="browsers">
<datalist id="browsers">
<option value="Internet Explorer">
<option value="Firefox">
<option value="Chrome">
<option value="Opera">
<option value="Safari">
</datalist>
<input type="submit">
</form>

Елемент <output>

Елемент output представляє результат обчислення (як той, що виконується скриптом)

<form action="action_page.php" oninput="x.value=parseInt(a.value)+parseInt(b/value)">
0
<input type="range" id="a" name="a" value="50">
100 +
<input type="number" id="b" name="b" value="50">
=
<output name="x" for="a b"></output>
<br/> <br/>
<input type="submit">
</form>

0 100 + =
